* 도면 내의 치수나 텍스트 숫자 값에 천 단위 구분 쉼표(,)를 자동으로 추가하거나 제거하는 기능입니다.

![][image1]

## **1\. 천 단위 구분하기** [**시연 영상**]()

1\) 명령창에 **“**CHTHOUSAND**”** 을 입력합니다.  
2\) 쉼표 처리를 적용할 텍스트 또는 치수 객체를 선택합니다.  
3\) 엔터 키를 누르면 선택한 객체의 숫자가 천 단위 구분 표시됩니다.

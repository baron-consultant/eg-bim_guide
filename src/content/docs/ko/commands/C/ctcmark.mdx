* 도면 내에서 철근 블록 위치를 기준으로 CTC 값을 생성 및 텍스트로 표기하는 명령어 입니다.

![][image1]

## **1\. 철근 CTC 표시하기** [**시연 영상**](https://youtu.be/EJuuV-MIpxs)

1\) 명령창에 “CTCMARK” 을 입력합니다.  
2\) 도면 내 철근 블록 선택합니다.  
3\) 철근 식별(RB\_NO) 값을 입력합니다.  
4\) CTC 마크가 표기됩니다.

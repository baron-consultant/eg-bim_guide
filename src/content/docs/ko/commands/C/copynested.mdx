- 블록 이나 외부 참조 내부의 객체를 선택하여 현재 도면으로 복사하는 기능입니다.

![][image1]

## **1\. 블록 객체를 일반 객체로 복사하기** [**시연 영상**](https://youtu.be/2YJbNnelqWM)

1\) 명령창에 “COPYNESTED” 명령어를 실행합니다.  
2\) 블록(Block), 외부 참조(Xref), DGN/PDF 언더레이 등에서 복사할 객체를 선택합니다.  
3\) 객체를 클릭하고 Ctrl \+ V 으로 붙여넣기 합니다.  
4\) 현재 도면의 작업 공간에 해당 객체가 독립된 객체로 복사됩니다.  
![][image2]

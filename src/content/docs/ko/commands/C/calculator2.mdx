- 도면 내 문자, 거리, 면적, 수식 등의 값을 계산하거나 변환하고 그 결과를 즉시 텍스트로 작성 또는 수정할 수 있는 고급 계산기 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/QuZZ8Ax_WGU)

1\) 명령창에 “CAL2” 또는 “CALCULATOR2” 입력합니다.  
2\) 아래 그림과 같은 계산기 대화상자가 나타납니다.  
![][image2]  
3\) 대화상자에서 계산 대상, 계산 방식, 소수 자리, 단위 변환 옵션 등을 설정합니다.  
4\) 작업이 완료되면 닫기를 클릭하여 종료합니다.

![][image1]

## **1\. 객체 방향 확인하기** [**시연 영상**](https://youtu.be/_C8gqWSocLg)

1\) 명령창에 “SCALEDISTANCE” 입력합니다.  
2\) 방향을 확인할 객체를 선택합니다.  
3\) 선택한 객체의 현재 방향이 객체에 표시됩니다.  
![][image2]

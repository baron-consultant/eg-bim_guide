* 도면 내의 치수 객체에 적용된 치수 축척 값을 기준 축척과 변경 축척 비율을 입력하여 일괄 변환하는 명령어입니다.

![][image1]

## **1\. 치수 비율 조정하기** [**시연 영상**](https://youtu.be/tDqr0Qwn19A)

1\) 명령창에 **“**CHDIMSCALE**”** 을 입력합니다.  
2\) 다음과 같은 순서로 명령창에 따라 값을 입력합니다.

- 기준 축척 값 입력 : 현재 도면의 기존 축척 값  
- 축척 비율 변경 값 입력 : 새로 적용할 축척 값

3\) 변경할 치수 객체를 선택합니다.  
4\) 엔터 키를 누르면 선택된 모든 치수 객체의 배율이 입력한 비율에 맞게 조정됩니다.

- 도면 내 객체의 좌표를 추출하여 엑세 파일로 내보내거나, 좌표 정보를 도면 상에 문자로 표시하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/z24F2KH7Pgc)

1\) 명령창에 “COORDINATESEXPORT” 입력합니다.  
2\) 표시된 설정창에서 원하는 옵션을 선택합니다.  
![][image2]  
3\) 좌표를 추출할 객체를 선택합니다.  
4\) 도면에 좌표가 표시되고 좌표가 엑셀 파일로 저장됩니다.  
![][image3]

- 도면 내 그룹을 생성, 수정, 관리할 수 있는 그룹 관리자 대화상자를 여는 기능입니다.

![][image1]

## **1\. 그룹 관리 열기** [**시연 영상**](https://youtu.be/iQPOm5Hekk0)

1\) 명령창에 “CLASSICGROUP” 명령어를 입력합니다.  
2\) Group Manager 대화상자가 나타납니다.  
![][image2]  
3\) 원하는 기능을 선택하여 그룹을 관리합니다.

## **2\. 주요 기능**

1\) Create New Group (새 그룹 생성)

- Name : 그룹 이름 지정
- Description : 그룹 설명 입력
- Selectable : 그룹을 객체처럼 선택할 수 있는지 여부 설정
- Create as unnamed group : 이름 없는 그룹으로 생성
- Select Entities and Create Group : 도면에서 객체 선택 후 그룹 생성

2\) Modify Selected Group (선택된 그룹 수정)

- Rename Group : 그룹 이름 변경
- Change Group Description : 그룹 설명 변경
- Reorder Entities : 그룹 내 객체 순서 변경
- Add Entities to Group : 그룹에 새로운 객체 추가
- Remove Entities from Group : 그룹에서 특정 객체 제거
- Ungroup Entities : 그룹 해제 (UNGROUP과 동일 기능)

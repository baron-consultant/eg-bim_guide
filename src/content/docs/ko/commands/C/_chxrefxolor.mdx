* 도면에 삽입된 외부참조(Xref) 객체의 표시 색상을 변경하는 명령어입니다.

![][image1]

## **1\. 외부참조 색상 변경하기** [**시연 영상**](https://youtu.be/0boyoua2xd0)

1\) 명령창에 “CHXREFXOLOR” 를 입력합니다.  
2\) 색상을 변경할 외부참조(Xref) 를 선택합니다.  
3\) 새로운 색상값(Color Index, TrueColor, 또는 이름) 을 입력합니다.

- 예: `1`(Red), `2`(Yellow)

4\) 엔터 키를 눌러 적용합니다.

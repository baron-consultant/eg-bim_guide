![][image1]

## **1\. 현재 레이어로 복사하기** [**시연 영상**](https://youtu.be/syvVSXDe4n4)

1\) 명령창에 “COPYTOCURL” 명령어를 입력합니다.  
2\) ENTER 키를 누른다.  
3\) 선택한 객체가 현재 설정된 레이어로 복사된다.  
4\) 원본 객체는 원래 레이어에 그대로 남고, 복사된 객체는 현재 레이어 속성을 따른다.  
![][image2]

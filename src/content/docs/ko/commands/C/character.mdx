- 특수문자 및 기호를 삽입할 수 있도록 문자표 창을 여는 기능입니다.

## **1\. 특수문자 입력하기** [**시연 영상**](https://youtu.be/S2ruafmB4xc)

1\) “CHARACTER” 명령어를 실행합니다.  
2\) 문자표 대화상자가 나타납니다.  
![][image1]  
3\) 입력할 특수문자 또는 기호를 선택합니다.  
4\) 선택(S) 또는 복사(C) 버튼을 클릭합니다.  
5\) MTEXT 편집기나 도면 내 텍스트 입력 시 Ctrl+V 로 붙여넣으면 문자가 삽입됩니다.

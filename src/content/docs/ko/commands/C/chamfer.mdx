- 꼭짓점이 존재하는 객체의 모서리를 따는 명령어입니다.

![][image1]

- Chamfer 옵션 키워드

* Polyline: 꼭짓점이 존재하는 2D 폴리선 객체를 선택하여 모든 코너에 모따기를 실행합니다.
* Distance: 하나의 꼭짓점에 연결된 두 변의 길이를 각각 설정합니다.
* Angle: 하나의 꼭짓점에 연결된 한 변의 길이를 정하고 그 변에서 인접한 다른 변까지의 각을 설정합니다.
* Method: Distance/Angle 중 어느 설정을 이용하여 모따기를 실행할지 설정합니다.  
  &nbsp;: Distance 키워드를 실행하고 나면 Distance 옵션으로,  
  &nbsp; Angle 키워드를 실행하고나면 Angle 옵션으로 자동으로 설정됩니다.
* Trim: 모서리를 딴 후 원본 코너를 삭제할지 여부를 설정합니다.  
   (No Trim: 삭제하지 않음, Trim: 삭제)
* Settings: Chamfer/Fillet 옵션 설정 CAD 팝업창을 엽니다.
* Multiple: ESC로 종료하기 전까지 모따기를 연속적으로 실행합니다.

![][image2]

- Chamfer 설정 팝업창

* Corners: 모서리를 딴 후 원본 코너를 삭제할지 여부를 설정합니다.
* Distance-distance: 하나의 꼭짓점에 연결된 두 변의 길이를 각각 설정해서 모따기를 실행합니다.
* Distance-angle: 하나의 꼭짓점에 연결된 한 변의 길이를 정하고 그 변에서 다른 변까지의 각을 설정해 모따기를 실행합니다.

## **1\. 객체의 모서리를 따는 방법** [**시연 영상**](https://youtu.be/XrSsD5VcH7k)

1\) “CHAMFER” 명령어를 실행합니다.  
2\) 모따기를 수행할 객체의 한 변을 선택합니다.  
3\) 선택한 변과 인접한 다른 변을 선택하면 모따기가 실행되고 명령어가 종료됩니다.

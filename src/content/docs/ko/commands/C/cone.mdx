- XY 평면에 평행한 3차원 원뿔을 그립니다.

## **1\. 반지름 지정 원뿔 그리기** [**시연 영상**](https://youtu.be/X-vIoY7Fzow)

1\) CONE 명령어를 실행합니다.  
2\) 원뿔 밑면의 중심이 될 좌표 선택 후 마우스 포인트 까지의 길이가 반지름인 원을 그립니다.  
3\) 높이를 지정하여 원뿔을 그립니다.

## **2\. 지름 지정 원뿔 그리기** [**시연 영상**](https://youtu.be/kEKskaiBUCw)

1\) CONE 명령어를 실행합니다.  
2\) 원뿔 밑면의 중심이 될 좌표 선택 후 명령어 창의 Diameter 옵션을 선택합니다.  
![][image1]  
3\) 마우스 포인트 까지의 길이가 지름인 원을 그립니다.  
4\) 높이를 지정하여 원뿔을 그립니다.

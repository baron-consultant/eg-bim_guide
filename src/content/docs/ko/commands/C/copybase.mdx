![][image1]

## **1\. 객체 복사(기준점 포함)하기** [**시연 영상**](https://youtu.be/HUXQQ25POKI)

1\) “COPYBASE” 명령어를 실행합니다.  
2\) 기준점(Base point) 을 먼저 지정합니다.

- 붙여넣을 때 이 점이 기준으로 사용됨.
- 객체의 꼭짓점, 중심점, 교차점 등 지정 가능.

3\) 복사할 객체(선, 폴리선, 원, 블록, 치수선 등)를 선택합니다.  
4\) ENTER를 눌러 선택을 완료합니다.  
5\) 선택한 객체가 지정한 기준점을 포함한 상태로 클립보드에 복사됩니다.  
 참고 : PASTECLIP 명령어를 실행하여 도면에 붙여넣습니다.

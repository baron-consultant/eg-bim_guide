- 현재 선택한 객체를 뷰포트를 기준으로 다른 공간(Model ↔ Paper)으로 이동합니다.

![][image1]

## **1\. 객체 공간 전환하기** [**객체 도면공간으로 이동 시연 영상**](https://youtu.be/iD7QMQcbgsk) [**객체 모델공간으로 이동 시연 영상**](https://youtu.be/bR84JRNwcq4)

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “CHSPACE” 를 입력합니다.  
3\) 이동할 객체를 선택합니다.  
4\) ENTER(Enter) 키를 누릅니다.  
5\) 선택한 객체가 자동으로 Model Space ↔ Paper Space 로 이동하며, 뷰포트 축척 에 따라 크기가 자동 조정됩니다.  
6\) 선택된 객체가 자동으로 배율이 조정되어 상대 공간으로 이동됩니다.

- `UNDO` 명령어로 취소한 작업을 다시 실행(되돌리기 취소)하는 기능입니다.

![][image1]

## **1\. 기본 실행** [**시연 영상**](https://youtu.be/5IhjzKUyEbE)

1\) “REDO” 명령어를 실행하거나, Ctrl \+ Y 단축키를 사용합니다.  
2\) 직전에 `UNDO`로 취소했던 작업이 다시 실행됩니다.  
3\) 여러 번 실행하면, `UNDO`로 취소한 여러 단계 작업을 순서대로 복구할 수 있습니다.

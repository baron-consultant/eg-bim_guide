* REFEDIT 명령으로 편집 중인 블록(Block) 또는 외부참조(Xref) 의 편집을 종료하는 명령어입니다.

![][image1]

## **1\. 블록 편집 종료하기** [**시연 영상**](https://youtu.be/v27v6u1tzDc)

1\) 명령창에 “REFEDIT” 를 입력하여 블록 또는 외부참조 편집을 시작합니다.  
2\) 편집할 객체를 수정합니다.  
3\) 명령창에 “REFCLOSE” 를 입력합니다.  
4\) 아래 두 가지 옵션 중 하나를 선택합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Save | 편집한 내용을 원본 블록 또는 외부참조 파일에 저장합니다. |
| Distcard | 변경 내용을 취소하고 원래 상태로 되돌립니다. |

5\) 블록 편집을 종료합니다.

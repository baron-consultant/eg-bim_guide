- 선택한 객체를 참조 각도를 기준으로 새로운 각도로 회전시키는 기능입니다.

![][image1]

## **1\. 객체 참조 회전** [**시연 영상**]()

1\) “REFROTATE” 명령어를 실행합니다.  
2\) 회전할 객체(선, 폴리선, 원, 사각형 등) 을 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 회전의 기준이 될 기준점을 지정합니다.  
5\) 참조 각도를 정의하기 위해 두 점을 지정하거나 값을 입력합니다.  
6\) 목표가 되는 새 각도를 입력합니다.  
7\) ENTER 또는 SPACE BAR를 누릅니다.  
8\) 선택한 객체가 참조 각도 대비 입력한 새 각도에 맞춰 회전됩니다.  
![][image2]

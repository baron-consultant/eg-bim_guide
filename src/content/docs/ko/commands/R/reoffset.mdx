- 선택한 객체를 동일한 간격으로 띄우는 명령어입니다.
- 선택 가능한 객체는 선, 폴리선, 호, 타원, 스플라인 등 선형 객체입니다.

 \- 간격 띄우기 전 \- \- 간격 띄우기 후 \-

## **1\. 동일 간격으로 여러번 간격 띄우기 \- Offset 개수 입력** [**시연 영상**](https://youtu.be/BpokTDptQlI)

1\) “REOFFSET” 명령어를 실행합니다.  
2\) 증분 Offset할 객체를 선택합니다.  
3\) 마우스로 Offset할 방향으로 점을 지정합니다.  
4\) 거리를 입력합니다.(값은 0 이하가 될 수 없습니다.)  
5\) Offset 개수를 입력합니다.

## **2\. 동일 간격으로 여러번 간격 띄우기 \- 두 점의 시작 점 지정** [**시연 영상**](https://www.youtube.com/watch?v=lscpdsDY0r0)

1\) “REOFFSET” 명령어를 실행합니다.  
2\) 증분 Offset할 객체를 선택합니다.  
3\) 마우스로 Offset할 방향으로 점을 지정합니다.  
4\) 거리를 입력합니다.(값은 0 이하가 될 수 없습니다.)  
5\) 두 점의 시작 점을 마우스로 지정합니다.

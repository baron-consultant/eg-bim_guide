* 수정되거나 덮어쓴 치수값을 원래 치수값으로 복원하는 기능입니다.

![][image1]

## **1\. 치수 복구하기** [**시연 영상**](https://youtu.be/YHd--yLtDHg)

1\) 명령창에 “RESTOREDIMVALUE” 를 입력합니다.  
2\) 치수 객체를 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 치수에 입력되어 있던 수동 텍스트가 삭제되고, 실제 치수값이 자동으로 다시 표시됩니다.

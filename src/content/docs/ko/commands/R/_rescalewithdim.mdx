* 도면 내의 객체를 치수 비율에 맞게 자동으로 스케일 조정하는 기능입니다.

![][image1]

## **1\. 치수 비율 조정하기** [**시연 영상**]()

1\) 명령창에 “RESCALEWITHDIM” 을 입력합니다.  
2\) 스케일을 조정할 객체를 선택합니다.  
3\) 기준 치수선을 클릭하여 실제 치수를 지정할 기준을 선택합니다.  
4\) 명령창에 실제 길이를 입력합니다. (예: 5000\)  
5\) 도면의 실제 치수값과 비교하여 자동으로 스케일이 계산되어 적용됩니다.  
6\) 명령창에 계산된 스케일 비율이 표시되고, 선택한 객체가 자동으로 확대 또는 축소됩니다.  
![][image2]

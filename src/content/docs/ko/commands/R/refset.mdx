* 외부참조 나 블록 편집 모드에서 현재 편집 중인 객체 외에 참조 표시되는 객체를 추가하거나 제거하는 명령어입니다.

![][image1]

## **1\. 편집 세트 구성하기** [**시연 영상**](https://youtu.be/uOuc5iQwUfk)

1\) 명령창에 “REFEDIT” 를 입력하여 블록 또는 Xref 편집을 시작합니다.  
2\) 편집 세트에 포함할 객체를 선택합니다.  
3\) 편집 도중 명령창에 “REFSET” 을 입력합니다.  
4\) 다음 중 하나의 옵션을 선택합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Add | 편집 세트에서 새 객체를 추가합니다. |
| Remove | 편집 세트에서 객체를 제거합니다. |

5\) 편집이 완료 되면 “REFCLOSE” 입력 후 변경 내용을 저장하고 확인합니다.

- 선택한 객체를 지정한 기준점을 중심으로 회전시키는 기능입니다.

![][image1]

## **1\. 객체 회전** [**시연 영상**](https://youtu.be/kHaDiJHMmD0)

1\) “ROTATE” 또는 “RO” 명령어를 실행합니다.  
2\) 회전할 객체(선, 폴리선, 원, 사각형 등)을 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 회전의 기준이 될 기준점을 지정합니다.  
5\) 회전할 각도를 입력합니다.

- 양수(+) → 반시계 방향 회전
- 음수(–) → 시계 방향 회전

6\) ENTER 또는 SPACE BAR를 누릅니다.  
7\) 선택한 객체가 입력한 각도만큼 기준점을 중심으로 회전됩니다.  
![][image2]

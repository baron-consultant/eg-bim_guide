* 사각형 형태의 폐쇄 철근에 점철근을 자동으로 배치하는 기능입니다.

![][image1]

## **1\. 사각 점철근 생성하기** [**시연 영상**](https://youtu.be/5CcvsQihrIQ)

1\) 명령창에 “RECREBAR” 를 입력합니다.  
2\) 명령창 안내에 따라 사각형 객체를 선택합니다.  
3\) 명령창 또는 대화상자에서 다음 항목을 설정합니다.

- Distance (D) : 점철근 간격  
- 내부\_외부(I) : 점철근을 외측 또는 내측 방향에 배치  
- 수평\_수직(H) : 점철근을 수평 또는 수직 방향에 배치  
- 시작 위치(P) : 점철근 시작 위치

4\) 설정이 완료되면, 선택한 사각형의 각 변을 따라 점철근이 자동으로 삽입됩니다.

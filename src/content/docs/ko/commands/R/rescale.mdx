- 도면에 삽입된 객체의 크기를 지정한 비율이나 단위에 맞게 다시 스케일 조정하는 기능입니다.

![][image1]

## **1\. 블록 축적 재지정하기** [**시연 영상**](https://youtu.be/HD8Z1NBy6hw)

1\) 명령창에 “RESCALE” 명령어를 입력한다.  
2\) 도면에서 크기를 변경할 객체(블록, 외부참조, 치수 등)를 선택한다.  
3\) 원하는 스케일 옵션을 입력한다.

- Scale Factor (배율 값) : 객체를 배율 값에 따라 확대/축소
- Units (단위 변환) : 예를 들어, Inches → Millimeters로 변환

4\) ENTER를 누르면 선택한 객체가 지정한 값에 따라 새 크기로 조정된다.

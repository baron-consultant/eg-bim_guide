title: RESETDIMTEXT
---

[image1]: ../../../../../assets/images/commands/resetdimtext/image1.png

* 이동되거나 수동으로 수정된 치수 텍스트를 원래 위치로 복귀시키는 기능입니다.

![][image1]

## **1\. 치수 위치 복원하기** [**시연 영상**](https://youtu.be/AyES5G8CC_U)

1\) 명령창에 “RESETDIMTEXT” 를 입력합니다.  
2\) 원래 위치로 복귀시킬 치수 객체를 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 선택된 치수의 텍스트가 자동으로 기준 위치(중앙) 로 복원됩니다.

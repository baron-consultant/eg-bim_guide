- 선택한 객체를 기준으로 외부 선분을 잘라내기 할 수 있습니다.

![][image1]

## **1\. 객체 외부 잘라내기** [**시연 영상**](https://youtu.be/_h69H1EF1N4)

1)“EXTENDTRIM” 명령어를 실행합니다.  
2\) 기준이 되는 객체를 선택합니다.  
3\) 삽입할 위치 좌표를 입력 합니다.  
4\) ENTER나 SPACE BAR를 눌러 종료합니다.  
![][image2]

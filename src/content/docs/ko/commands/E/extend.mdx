- 선택한 경계 객체를 기준으로 선, 폴리선, 호 등의 객체를 경계까지 연장하는 명령어입니다.

![][image1]

## **1\. 객체 연장하기** [**시연 영상**](https://youtu.be/_h69H1EF1N4)

1\) “EXTEND” 명령어를 실행합니다.  
2\) 연장의 기준이 될 경계 객체(목표선)를 선택합니다.  
3\) ENTER를 누릅니다.  
4\) 연장할 객체를 클릭하면 기준선까지 자동으로 연장됩니다.  
5\) ENTER나 SPACE BAR를 눌러 종료합니다.  
![][image2]

- 도면에서 텍스트(Text) 와 점(Point) 을 한 쌍으로 자동 생성하는 명령어입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/j-s6OboTwNA)

1\) 명령창에 “ELVALUETOTEXT” 를 입력합니다.  
2\) 대화상자에서 필요한 항목을 설정합니다.  
![][image2]  
3\) 선택한 객체의 Z 좌표 값을 자동으로 읽습니다.  
4\) 해당 위치(또는 설정된 오프셋 위치)에 EL 텍스트가 생성됩니다.

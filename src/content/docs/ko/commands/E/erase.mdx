- 도면에서 선택한 객체를 삭제하는 기능입니다.

![][image1]

## **1\. 객체 삭제** [**시연 영상**](https://youtu.be/PefIXL8dNfE)

1\) “ERASE” 또는 “E” 명령어를 실행합니다.  
2\) 삭제할 객체(선, 폴리선, 원, 블록, 해치, 이미지 등)를 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 선택한 객체가 도면에서 삭제됩니다.  
![][image2]

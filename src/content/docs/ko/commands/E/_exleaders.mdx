* 기준 지시선을 기준 객체에 맞춰 연장하거나 축소(끊기) 하는 명령어입니다.

![][image1]

## **1\. 지시선 연장 / 축소하기** [**시연 영상**]()

1\) 명령창에 “EXLEADERS” 를 입력합니다.  
2\) 기준 객체를 선택합니다.  
3\) 연장 또는 축소를 적용할 지시선을 선택합니다.  
4\) 엔터 키를 누릅니다.  
5\) 선택한 객체를 기준으로 지시선이 자동으로 연장 또는 축소됩니다.

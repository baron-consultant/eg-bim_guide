- 도면에서 객체 종류와 선택 규칙(영역/교차/경계선) 을 한꺼번에 지정해 해당 조건에 맞는 객체만 빠르게 선택하는 고급 “빠른 선택” 도구입니다.

![][image1]

## **1\. 특정 객체 선택하기** [**시연 영상**]()

1\) 명령창에 “QSELECT2” 를 입력합니다.  
2\) 나타난 대화상자에서 필요한 사항을 설정합니다.  
![][image2]  
3\) 객체를 선택하면 해당 조건의 객체만 선택됩니다.

- 도면에서 특정 조건(예: 레이어, 색상, 객체 종류 등)을 기준으로 객체를 빠르게 선택 하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) 명령창에 “QSELECT” 를 입력합니다.  
2\) 빠른 선택 대화상자가 나타납니다.  
![][image2]  
3\) 대화상자에서 필요한 항목을 선택합니다.  
4\) 설정이 완료되면 조건에 맞는 객체를 빠르게 선택합니다.

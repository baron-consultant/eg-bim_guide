- 도면 내의 모든 문자를 빠른 텍스트 표시로 단순화하여 보여주는 기능입니다.

![][image1]

## **1\. 문자 표시상태 설정하기** [**시연 영상**](https://youtu.be/JhnMX0BsOQk)

1\) “QTEXT” 명령어를 실행합니다.  
2\) 표시 설정 여부를 입력합니다:

- ON : 문자를 실제 텍스트 대신 사각형(또는 단순 프레임)으로 표시
- OFF : 문자를 정상적으로 표시 (기본값)

3\) ENTER 또는 SPACE BAR를 누르면 설정이 적용됩니다.  
 참고 : “REGEN” 명령어를 사용하여 새로고침해야 화면에 반영됩니다.  
![][image2]

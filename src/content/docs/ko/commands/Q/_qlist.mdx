- 선택한 객체의 길이(Length)와 면적(Area) 을 빠르게 조회하는 명령어입니다.

![][image1]

## **1\. 객체 정보 조회하기** [**시연 영상**]()

1\) 명령창에 “QLIST” 입력합니다.  
2\) 길이 또는 면적을 확인할 객체를 선택합니다.  
3\) ENTER 키를 누릅니다.  
4\) 명령창에 선택한 객체의 길이와 면적이 표시됩니다.  
![][image2]

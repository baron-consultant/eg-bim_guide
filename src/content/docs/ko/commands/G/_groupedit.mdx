- 이미 만들어진 그룹의 객체를 추가하거나 제거할 수 있는 기능입니다.
- 그룹을 해제하지 않고도 구성 요소를 수정할 수 있습니다.

![][image1]

## **1\. 그룹 편집하기**

1\) 명령창에 “GROUPEDIT” 명령어를 입력합니다.  
2\) 수정할 그룹 선택 또는 그룹 이름을 입력합니다.  
3\) 다음 옵션 중 하나를 입력합니다.

- Add : 그룹에 새 객체를 추가
- Remove : 그룹에서 특정 객체를 제거
- ? : 현재 그룹에 포함된 이름 확인
- reName : 그룹 이름 변경

4\) ENTER 또는 SPACE BAR를 누르면 변경 사항이 적용됩니다.

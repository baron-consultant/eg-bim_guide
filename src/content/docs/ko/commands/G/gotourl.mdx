## **1\. 객체 하이퍼링크 추가** [**시연 영상**](https://youtu.be/dbSBQFoNXvU)

1\) “GOTOURL” 명령어를 실행합니다.  
2\) 하이퍼링크가 설정된 객체(선, 폴리선, 블록, 이미지 등)를 선택합니다.  
3\) 객체에 포함된 링크가 즉시 실행됩니다.

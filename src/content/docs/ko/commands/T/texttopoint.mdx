- 도면에 존재하는 텍스트 객체의 좌표값을 자동으로 인식하여 해당 위치에 점 또는 기호 를 생성하는 명령어입니다.

![][image1]

## **1\. 좌표 생성하기** [**시연 영상**](https://youtu.be/W_YjmqgPhfI)

1\) 명령창에 “TEXTTOPOINT” 를 입력합니다.  
2\) 대화상자에서 필요한 항목을 설정합니다.  
![][image2]  
3\) 명령이 자동으로 각 텍스트의 내용을 분석하여 숫자 형태의 좌표(X, Y, Z)를 인식합니다.  
4\) 인식된 좌표값을 기준으로 해당 위치에 점 객체가 생성됩니다.

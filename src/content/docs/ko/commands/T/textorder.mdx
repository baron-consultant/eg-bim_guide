- 내용(문자/숫자 값)을 기준으로 오름차순 또는 내림차순 정렬하고, 지정한 방향 순서에 따라 자동으로 재배치하는 기능입니다.

![][image1]

## **1\. 텍스트 정렬하기** [**시연 영상**](https://youtu.be/WgMGUGmAPlg)

1\) “TEXTORDER” 명령어를 실행합니다.  
2\) “S” 를 입력하고 TEXTORDER 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 정렬할 여러 텍스트 객체를 선택합니다.  
4\) ENTER 또는 SPACE BAR를 누릅니다.  
5\) 선택한 텍스트가 지정한 정렬 기준과 배열 순서에 따라 자동 정렬됩니다.  
![][image3]

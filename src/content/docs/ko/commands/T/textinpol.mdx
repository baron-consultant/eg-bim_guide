- 선택한 폴리곤(폐합 폴리선, 도형) 내부에 텍스트를 자동으로 배치하는 기능입니다.

![][image1]

## **1\. 텍스트 배치** [**시연 영상**](https://youtu.be/yfXmHdjPy5M)

1\) “TEXTINPOL” 명령어를 실행합니다.  
2\) TEXTINPOL 대화상자 창에서 원하는 옵션을 선택합니다.  
![][image2]  
3\) 도면에서 텍스트를 배치할 지점을 선택합니다.  
4\) 배치가 끝나면 ESC 키를 눌러 종료합니다.  
![][image3]

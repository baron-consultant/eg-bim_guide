- 문자 객체가 출력될 때 채워진 문자로 표시할지, 윤곽선만 표시할지를 설정하는 기능입니다.

![][image1]

## **1\. 문자 채움 표시 설정하기** [**시연 영상**](https://youtu.be/wcyPGhHWEpQ)

1\) “TEXTFILL” 명령어를 실행합니다.  
2\) 옵션 값을 입력합니다.

- ON : 문자 내부가 채워져 표시됨 (기본값)
- OFF : 문자 외곽선만 표시됨

3\) ENTER 또는 SPACE BAR를 누르면 설정이 적용됩니다.  
 참고 : “REGEN” 명령어를 사용하여 새로고침해야 화면에 반영됩니다.  
![][image2]

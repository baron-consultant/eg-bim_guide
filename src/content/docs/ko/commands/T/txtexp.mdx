- 문자 객체를 개별 폴리선 또는 선분 으로 변환하는 기능입니다.

![][image1]

## **1\. 문자 객체 분해하기** [**문자 외곽선 추출 시연 영상**](https://youtu.be/6so3K_mhmbc)

1\) 명령창에 “TXTEXP” 명령어를 실행합니다.  
2\) 도형으로 변환할 문자 객체를 선택합니다.  
3\) ENTER 또는 SPACE BAR를 누릅니다.  
4\) 해당 문자가 선, 호, 폴리선 으로 분해되어 생성됩니다.  
![][image2]

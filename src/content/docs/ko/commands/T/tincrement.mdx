- 텍스트를 자동으로 순차 증가시켜 일련번호, 도면번호, 부품번호, 좌표점 번호 등을 빠르게 작성할 수 있는 기능입니다.

![][image1]

## **1\. 번호 자동 증가시키기** [**시연 영상**](https://youtu.be/LuHzOsUZZPs)

1\) 명령창에 “TINCREMENT” 입력합니다.  
2\) 명령창에 S 를 입력하면 설정 대화상자가 나타납니다.  
![][image2]  
3\) 대화상자에서 필요한 옵션을 설정합니다.  
4\) 숫자가 포함된 객체를 선택합니다.  
명령창 또는 대화상자에서 다음 항목을 설정합니다.  
5\) 복사 기준점을 선택합니다.  
6\) 화면에서 텍스트를 삽입할 위치를 클릭합니다.  
7\) 클릭할 때마다 번호가 자동 증가하면서 새로운 텍스트가 생성됩니다.  
8\) 종료하려면 ENTER 키를 누릅니다.  
![][image3]

* 선택한 기준 문자의 속성(글꼴, 높이, 색상, 정렬 등)을 다른 문자에 복사하여 동일하게 적용하는 명령입니다.

![][image1]

## **1\. 문자 속성 복사하기** [**시연 영상**]()

1\) 명령창에 “TEXTMATCHCOPY” 를 입력합니다.  
2\) 기준이 될 원본 문자(Text 또는 Mtext) 를 선택합니다.  
3\) 동일한 속성을 적용할 대상 문자(하나 이상) 를 선택합니다.  
4\) 엔터 키를 누릅니다.  
5\) 원본 문자와 동일한 속성(글꼴, 높이, 색상, 정렬 등)이 속성이 대상 문자에 복사되어 적용됩니다.

- 도면 내의 문자 객체를 추출하여 엑셀 파일로 내보내는 기능입니다.

![][image1]

## **1\. 문자 → 엑셀 내보내기** [**시연 영상**](https://youtu.be/7JNcoqOGGXw)

1\) “TEXTTOEXL” 명령어를 실행합니다.  
2\) TEXTTOEXL 대화상자에서 원하는 옵션을 설정합니다.  
![][image2]  
3\) 엑셀로 내보낼 문자 객체를 선택합니다.  
4\) ENTER 또는 SPACE BAR를 눌러 선택을 완료합니다.  
5\) 엑셀이 자동으로 실행되며 시작 셀을 선택합니다.  
6\) 확인을 누르면 선택한 문자 내용이 엑셀 시트에 저장됩니다.  
![][image3]

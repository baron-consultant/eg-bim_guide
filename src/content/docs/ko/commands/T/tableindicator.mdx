- 도면에 삽입된 표의 행과 열 번호 표시 여부를 설정하는 기능입니다.

![][image1]

## **1\. 행,열 번호 표시 설정하기** [**시연 영상**](https://youtu.be/pbuYHs_VHCM)

1\) 명령창에 “TABLEINDICATOR`”` 명령어를 실행합니다.  
2\) 행/열 번호 표시 여부를 묻는 메시지가 나타납니다.

- 1 : 표의 상단과 좌측에 열 번호(A, B, C…) 와 행 번호(1, 2, 3…) 가 표시됨
- 0 : 번호 표시가 숨겨져 일반 표 형태로만 보임

3\) Enter를 눌러 설정을 적용합니다.

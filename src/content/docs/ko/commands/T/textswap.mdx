- 선택한 두 텍스트 객체의 문자 내용을 서로 교환(스왑) 하는 기능입니다.

![][image1]

## **1\. 문자내용 교환하기** [**시연 영상**](https://youtu.be/eNlZng0NCTo)

1\) “TEXTSWAP” 명령어를 실행합니다.  
2\) 문자 내용을 교환할 첫 번째 텍스트 객체를 선택합니다.  
3\) 문자 내용을 교환할 두 번째 텍스트 객체를 선택합니다.  
4\) ENTER 또는 SPACE BAR를 누르면 두 텍스트의 문자 위치가 서로 바뀝니다.  
![][image2]

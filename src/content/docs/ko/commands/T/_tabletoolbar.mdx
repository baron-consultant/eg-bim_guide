- 도면에 삽입된 표를 편집할 때 나타나는 표 편집 도구막대의 표시 여부를 제어하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) 명령창에 “TABLETOOLBAR`”` 명령어를 실행합니다.  
2\) 표시 여부를 설정합니다.

- ON : 표를 선택하거나 편집할 때 Table Toolbar 가 화면에 표시됨
- OFF : 표를 선택해도 Toolbar가 표시되지 않음

3\) 설정을 저장하면 이후 표 편집 시 적용됩니다.

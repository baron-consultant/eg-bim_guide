- 도면에서 텍스트(Text) 와 점(Point) 을 한 쌍으로 자동 생성하는 명령어입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) 명령창에 “TEXTANDPOINT” 를 입력합니다.  
2\) 대화상자에서 필요한 항목을 설정합니다.  
![][image2]  
3\) 화면에서 좌표를 클릭합니다.  
4\) 클릭한 지점에 점과 해당 좌표 또는 EL값의 텍스트가 동시에 생성됩니다.

- 선택한 문자 객체를 사용자가 지정한 기준점을 중심으로 원하는 각도만큼 회전시키는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/kXvycWOuXho)

1\) 명령창에 “TEXTROTATE” 명령어를 실행합니다.  
2\) 회전할 문자 객체를 선택합니다.  
3\) 첫 번째 기준점(회전 시작점)을 지정합니다.  
4\) 두 번째 기준점(회전 방향)을 지정합니다.  
5\) Enter를 누르면 선택한 문자가 지정한 두 점을 기준으로 회전합니다.  
![][image2]

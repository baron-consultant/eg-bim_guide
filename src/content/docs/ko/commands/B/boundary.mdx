- 닫힌 영역을 인식해서 폴리라인(Polyline) 또는 리전(Region)으로 자동 생성하는 기능입니다.  
  ![][image1]
- Select boundary set 버튼 클릭 시 경계를 찾을 대상 객체의 범위를 지정할 수 있습니다.
- All visible entities 옵션 선택 시 현재 화면에 보이는 모든 객체를 기준으로 경계를 검색합니다.
- Select boundary set로 미리 대상 객체가 선택되어있을 경우 Current selection set 옵션이 선택되고 boundary set내에서만 영역을 선택할 수 있습니다.
- Island detection옵션이 켜져있는 경우 경계 내부에 있는 또 다른 닫힌 영역을 모두 인식해 뚫린 모양대로 경계를 만들어줍니다. 체크 해제 시 내부의 닫힌 영역을 무시하고 가장 바깥쪽 영역만 선택됩니다.

## **1\. 경계 또는 영역 작성하기** [**시연 영상**](https://youtu.be/186Y8qZpF28)

1\) 명령어를 실행하면 팝업창이 실행됩니다.  
2\) Pick internal points 버튼 또는 OK 버튼 클릭 시 영역 선택 동작이 실행됩니다.  
3\) 선택하고 싶은 영역의 내부를 마우스로 클릭합니다. 클릭한 영역의 외곽선이 하이라이트됩니다.  
4\) 선택이 끝나면 ENTER를 입력합니다. 기존 선 위로 Entity type 옵션값에 따라 Polyline 또는 Region이 생성됩니다.

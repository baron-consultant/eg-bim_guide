- 객체 위에서 지정한 점 위치를 기준으로 객체를 끊어내는 기능입니다

![][image1]

## **1\. 객체 끊어내기** [**시연 영상**](https://youtu.be/uBROCy2oC4A)

1\) “BRESDIST” 명령어를 실행합니다.  
2\) “S” 를 입력하여 객체를 끊을 거리, 간격 등을 설정 합니다.  
3\) 객체(선, 폴리선, 사각형 등)를 선택합니다.  
4\) ENTER 또는 SPACE BAR를 누릅니다.  
5\) 선택한 객체는 동일한 간격으로 나누어 집니다.  
![][image2]

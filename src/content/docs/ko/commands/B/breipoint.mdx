* 선택한 객체(선, 폴리선, 호 등)를 하나의 기준점에서 끊어내어 두 개의 별도 객체로 분리합니다.

![][image1]

## **1\. 객체 끊어내기(객체 기준)** [**시연 영상**](https://youtu.be/W5O15QkB_sI)

1\) 명령창에 “BREIPOINT”를 입력합니다.  
2\) 끊을 기준 객체(선, 폴리선, 호 등)를 선택합니다.  
3\) 끊을 객체를 선택합니다.  
4\) 선택한 객체를 기준으로 객체가 두 개의 독립된 객체로 분리됩니다.

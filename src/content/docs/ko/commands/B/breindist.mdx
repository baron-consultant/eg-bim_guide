- 선택한 객체의 입력한 거리 만큼 끊어내는 기능입니다.

![][image1]

## **1\. 객체 끊어내기** [**시연 영상**](https://youtu.be/EG7jhVRcUQU)

1\) “BREINDIST” 명령어를 실행합니다.  
2\) 객체(선, 폴리선, 사각형 등)를 선택합니다.  
3\) 시작 점을 선택하고 끊을 거리를 입력 합니다.  
![][image2]

* 배열 객체(Array Object) 를 편집할 수 있도록 편집 모드로 진입하는 명령입니다. 이 명령을 사용하면 배열의 개수, 간격, 각도, 방향 등 구성 요소를 수정할 수 있습니다.

![][image1]

## **1\. 배열 편집하기** [**시연 영상**](https://youtu.be/aPo2uWuuvck)

1\) 명령창에 “ARRAYEDIT” 를 입력합니다.  
2\) 편집할 배열 객체(Array) 를 선택합니다.  
3\) 다음 옵션 중 하나를 선택합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Source	 | 배열의 원본 객체를 편집합니다. 원본을 수정하면 전체 배열에 반영됩니다. |
| REPlace | 배열 내 특정 항목을 다른 객체로 교체합니다. |
| Base point | 배열의 기준점을 변경합니다. 배열의 기준 위치가 바뀝니다. |
| Rows | 배열의 행 개수 및 간격을 변경합니다. |
| Columns | 배열의 열개수 및 간격을 변경합니다. |
| Levels | 3D 배열에서 층 개수 및 높이 간격을 조정합니다. |
| RESet | 현재 편집 내용을 초기화하고, 원래 배열 설정으로 되돌립니다. |
| eXit | 배열 편집 모드를 종료합니다. |

4\) 필요한 변경을 완료합니다.

* 선택한 객체를 지정한 중심점을 기준으로 원형 형태로 일정 각도 간격 또는 전체 각도 내에 균등하게 복제하는 기능입니다.

![][image1]

## **1\. 원형 배열 생성** [**시연 영상**](https://youtu.be/QO8_I3KT6VY)

1\) 명령창에 “ARRAYPOLAR” 를 입력합니다.  
2\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
3\) 원형 배열의 중심점을 선택하고 엔터 키를 누릅니다.  
4\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Base point | 배열의 기준점을 재지정할 수 있습니다. 기본은 선택한 객체의 중심입니다. |
| Items | 배열할 객체의 개수 를 입력합니다. |
| Angle between | 각 객체 간의 간격 각도(°) 를 지정합니다. |
| Fill angle | 전체를 채울 총 각도(°) 를 지정합니다. 기본값은 360°입니다. (예: 180° → 반원 형태 배열) |
| Rows | 배열을 다층(원형 중첩) 형태로 만들 때 사용합니다. (기본 1\) |
| Levels | Z축 방향으로 여러 층으로 배열할 수 있습니다. (3D 작업용) |
| Rotate items | 각 객체가 중심을 향해 회전 정렬될지 여부를 지정합니다. \- ON : 객체가 중심에 대해 회전 (기본값) \- OFF : 객체 방향 유지 |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 지정한 중심점을 따라서 배열이 생성됩니다.

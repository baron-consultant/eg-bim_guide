* 지정한 경로를 따라 객체를 일정한 간격 또는 개수 기준으로 배열 복사하는 명령어입니다.

![][image1]

## **1\. 선형 배열 생성** [**시연 영상**](https://youtu.be/YUQPeeqajhU)

1\) 명령창에 “ARRAYPATH” 를 입력합니다.  
2\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
3\) 배열할 경로를 선택합니다.  
4\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Method | 배열 기준 방식을 선택합니다. \- Divide : 개수를 기준으로 배열 \- Measure : 간격을 기준으로 배열 |
| Base point | 복제 시 사용할 기준점을 지정합니다. |
| Tangent direction | 배열된 객체의 방향을 경로의 접선 방향으로 맞출지 설정합니다. |
| Items | 배열할 객체의 개수 를 입력합니다. |
| Rows | 여러 줄로 배열할 때 행 수 를 지정합니다. (기본 1\) |
| Levels | Z축 방향으로 여러 층으로 배열할 수 있습니다. (3D 작업용) |
| Align items | 객체를 경로 방향에 정렬할지 여부를 설정합니다. \- ON : 경로의 방향에 맞춰 자동 회전 \- OFF : 원래 각도 유지 |
| Z direction | Z축 방향 정렬을 켜거나 끕니다. 3D 경로에서 객체 방향을 제어할 때 사용합니다. |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 지정한 경로를 따라서 배열이 생성됩니다.

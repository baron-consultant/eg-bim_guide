- 선택한 텍스트 객체에 머리말, 중간말, 꼬리말을 손쉽게 추가하는 기능입니다.

![][image1]

## **1\. 텍스트 편집** [**시연 영상**](https://youtu.be/EET3S3sJgbo)

1\) “APTEXT” 명령어를 실행합니다.  
2\) 삽입할 객체를 선택합니다.  
3\) APTEXT 대화상자에서 필요한 항목을 입력합니다.  
![][image2]

- 머리말 : 텍스트 맨 앞에 추가할 문자열
- 중간말 : 지정한 위치에 삽입할 문자열
- 꼬리말 : 텍스트 맨 끝에 추가할 문자열
- 중간말 입력 위치 : 원본 문자열에서 중간말을 삽입할 위치(문자 인덱스, 숫자)

4\) 설정이 끝나면 확인 버튼을 클릭합니다.  
5\) 선택한 텍스트에 지정한 머리말/중간말/꼬리말이 반영됩니다.  
![][image3]

* 도면 내의 문자 또는 속성 객체에 자동으로 표고(Elevation, EL 값) 를 적용하거나 업데이트하는 명령어입니다.

![][image1]

## **1\. 문자 표고(Elevation) 값을 객체에 적용하기** [**시연 영상**](https://youtu.be/o7g1xPP9JH0)

1\) 명령창에 “APPLYELTOTEXT” 을 입력합니다.  
2\) Z 값을 적용할 숫자 문자를 선택 합니다.  
3\) Z 값을 참조할 객체 을 클릭합니다.  
4\) 엔터 키를 누릅니다.  
5\) 선택한 객체의 Z값이 선택한 문자의 값으로 자동 변경됩니다.

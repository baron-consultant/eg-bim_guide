- 외부 파일을 현재 도면에 참조(Attach) 형태로 삽입하는 기능입니다.
- 지원되는 파일은 도면 파일(DWG), 이미지 파일, DWF, PDF, DGN 등입니다.

![][image1]

## **1\. 외부 파일 블록으로 부착하기** [**시연 영상**](https://youtu.be/fhoR2jzEtQU)

1\) 명령창에 “ATTACH” 명령어를 실행합니다.  
2\) 파일 선택 대화상자가 나타납니다.

- 삽입할 DWG, DWF, PDF, 이미지(JPG, PNG, BMP 등), DGN 파일을 선택

![][image2]  
3\) 확인하면 선택한 파일이 도면에 외부 참조로 삽입됩니다.

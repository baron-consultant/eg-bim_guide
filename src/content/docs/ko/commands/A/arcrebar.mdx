* 원호 위에 점철근을 삽입하는 기능입니다.

![][image1]

## **1\. 원 점철근 생성하기** [**시연 영상**](https://youtu.be/zmrX6HCM204)

1\) 명령창에 “ARCREBAR” 를 입력합니다.  
2\) 점철근을 배치할 원호(Arc) 또는 폴리라인 곡선 구간을 클릭합니다.  
3\) 명령창 또는 대화상자에서 다음 항목을 설정합니다.

- Count (C) : 점철근 개수를 직접 지정  
- Scale (S) : 점철근의 크기를 직접 지정  
- 외측(I) / 내측(O) : 점철근을 외측 또는 내측 방향에 배치할지 설정

4\) 설정이 완료되면, 선택한 원호를 따라 점철근이 삽입됩니다.

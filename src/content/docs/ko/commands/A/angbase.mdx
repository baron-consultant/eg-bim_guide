- 도면에서 각도의 기준 방향을 설정하는 명령어입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/9zdDb6dWxxY)

1\) 명령창에 “ANGBASE” 를 입력합니다.  
2\) 명령창에서 기준 각도(Base angle) 값을 입력합니다.

- 0 \= 동쪽(East) 0° 기준
- 90 \= 북쪽(North) 0° 기준
- 180 \= 서쪽(West) 0° 기준
- 270 \= 남쪽(South) 0° 기준

3\) ENTER를 눌러 설정을 적용합니다.

- 선택한 영역의 면적(Area) 과 둘레 길이(Perimeter) 를 계산하는 명령어입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/GyTnsDbHzeI)

1\) 명령창에 “AREA” 입력합니다.  
2\) ENTER 키를 누릅니다.  
3\) 명령창의 옵션 중 하나를 선택합니다.

- Entity : 특정 객체를 선택하여 면적 계산
- Add : 여러 영역을 더하여 합산
- Subtract : 특정 영역을 빼서 순면적 계산

4\) 계산이 완료되면 명령창에 면적(Area)과 둘레(Perimeter)가 표시됩니다.  
 참고 : 객체의 점을 클릭하면 임의의 다각형 영역 지정하여 영역이 계산 됩니다.

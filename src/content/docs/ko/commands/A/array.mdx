* 선택한 객체를 일정한 규칙에 따라 여러 개 복사(배열) 하는 기능입니다.

![][image1]

## **1\. 배열 사용 방법**

1\) 명령창에 “ARRAY” 를 입력합니다.  
2\) 배열할 객체를 선택하고 Enter 키를 누릅니다.  
3\) 명령창에서 배열 형태를 선택합니다.

- Rectangular (직사각형 배열)  
- Path (경로 배열)  
- Polar (원형 배열)

## **2\. 직사각형 배열 사용 방법** [**시연 영상**](https://youtu.be/04KPr6xtJ0A)

1\) 명령창에 “Rectangular” 옵션을 선택합니다.  
1\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
2\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Base point | 배열의 기준점을 지정합니다. 이 점을 중심으로 행·열 방향으로 복제됩니다. |
| COunt | 전체 복제 개수를 지정합니다.  (열×행×층의 전체 개수 대신 직접 개수 지정 가능) |
| Base point | 배열의 기준점을 지정합니다. 이 점을 중심으로 행·열 방향으로 복제됩니다. |
| Spacing | 복제 간격을 지정합니다. 각 객체 간의 가로 및 세로 거리를 수동으로 입력 가능 |
| COLumns | 가로 방향(열)의 개수를 설정합니다. |
| Rows | 세로 방향(행)의 개수를 설정합니다. |
| Levels | Z축(높이) 방향으로 층을 추가할 수 있습니다. (3D 모델에서 사용) |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 선택한 객체를 기준으로 배열이 생성됩니다.

## **3\. 경로 배열 사용 방법** [**시연 영상**](https://youtu.be/BDAKVVPST5A)

1\) 명령창에 “Path ” 옵션을 선택합니다.  
2\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
3\) 배열할 경로를 선택합니다.  
4\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Method | 배열 기준 방식을 선택합니다. \- Divide : 개수를 기준으로 배열 \- Measure : 간격을 기준으로 배열 |
| Base point | 복제 시 사용할 기준점을 지정합니다. |
| Tangent direction | 배열된 객체의 방향을 경로의 접선 방향으로 맞출지 설정합니다. |
| Items | 배열할 객체의 개수 를 입력합니다. |
| Rows | 여러 줄로 배열할 때 행 수 를 지정합니다. (기본 1\) |
| Levels | Z축 방향으로 여러 층으로 배열할 수 있습니다. (3D 작업용) |
| Align items | 객체를 경로 방향에 정렬할지 여부를 설정합니다. \- ON : 경로의 방향에 맞춰 자동 회전 \- OFF : 원래 각도 유지 |
| Z direction | Z축 방향 정렬을 켜거나 끕니다. 3D 경로에서 객체 방향을 제어할 때 사용합니다. |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 지정한 경로를 따라서 배열이 생성됩니다.

## **4\. 원형 배열 사용 방법** [**시연 영상**](https://youtu.be/2p-fUpDUftE)

1\) 명령창에 “Polar” 옵션을 선택합니다.  
2\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
3\) 원형 배열의 중심점을 선택하고 엔터 키를 누릅니다.  
4\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Base point | 배열의 기준점을 재지정할 수 있습니다. 기본은 선택한 객체의 중심입니다. |
| Items | 배열할 객체의 개수 를 입력합니다. |
| Angle between | 각 객체 간의 간격 각도(°) 를 지정합니다. |
| Fill angle | 전체를 채울 총 각도(°) 를 지정합니다. 기본값은 360°입니다. (예: 180° → 반원 형태 배열) |
| Rows | 배열을 다층(원형 중첩) 형태로 만들 때 사용합니다. (기본 1\) |
| Levels | Z축 방향으로 여러 층으로 배열할 수 있습니다. (3D 작업용) |
| Rotate items | 각 객체가 중심을 향해 회전 정렬될지 여부를 지정합니다. \- ON : 객체가 중심에 대해 회전 (기본값) \- OFF : 객체 방향 유지 |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 지정한 중심점을 따라서 배열이 생성됩니다.

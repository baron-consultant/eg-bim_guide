* 선택한 객체를 가로, 세로, 높이 방향으로 일정 간격을 두고 격자 형태로 복제하는 명령어입니다.

![][image1]

## **1\. 직사각형 배열 생성** [**시연 영상**](https://youtu.be/EdXQm93DFBo)

1\) 명령창에 “ARRAYRECT” 를 입력합니다.  
2\) 배열할 객체를 선택하고 엔터 키를 누릅니다.  
4\) 명령창 또는 대화상자에서 필요한 옵션을 설정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Associative | 배열을 하나의 그룹으로 유지할지 설정합니다. \- ON : 배열된 객체를 하나의 연관 배열로 유지 (기본값) \- OFF : 개별 객체로 분리 생성됨 |
| Base point | 배열의 기준점을 지정합니다. 이 점을 중심으로 행·열 방향으로 복제됩니다. |
| COunt | 전체 복제 개수를 지정합니다.  (열×행×층의 전체 개수 대신 직접 개수 지정 가능) |
| Base point | 배열의 기준점을 지정합니다. 이 점을 중심으로 행·열 방향으로 복제됩니다. |
| Spacing | 복제 간격을 지정합니다. 각 객체 간의 가로 및 세로 거리를 수동으로 입력 가능 |
| COLumns | 가로 방향(열)의 개수를 설정합니다. |
| Rows | 세로 방향(행)의 개수를 설정합니다. |
| Levels | Z축(높이) 방향으로 층을 추가할 수 있습니다. (3D 모델에서 사용) |
| eXit | 배열 설정을 완료하고 명령을 종료합니다. |

5\) 선택한 객체를 기준으로 배열이 생성됩니다.

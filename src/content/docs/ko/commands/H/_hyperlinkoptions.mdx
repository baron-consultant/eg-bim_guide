## **1\. 하이퍼링크 표시 옵션 설정하기** [**시연 영상**]()

1\) 명령창에 “HYPERLINKOPTIONS” 를 입력합니다.  
2\) 첫 번째 옵션을 설정합니다.

- Display hyperlink cursor and shortcut menu?  
  (하이퍼링크 아이콘 표시 및 우클릭 메뉴 표시 여부)  
- Yes : 링크 커서와 바로가기 메뉴 표시  
- No : 표시하지 않음

3\) 두 번째 옵션을 설정합니다.

- Display hyperlink tooltips? (마우스를 올렸을 때 링크 주소(툴팁)를 표시할지 여부)  
- Yes : 링크 주소 툴팁 표시  
- No : 표시하지 않음

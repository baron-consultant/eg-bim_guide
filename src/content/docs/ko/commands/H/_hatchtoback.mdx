* 도면 내 해칭(Hatch) 객체를 모든 다른 객체 뒤로 이동(배치) 시키는 명령어입니다.

![][image1]

## **1\. 해치 맨 뒤로 보내기** [**시연 영상**]()

1\) 명령창에 “HATCHTOBACK” 를 입력합니다.  
2\) 명령이 자동으로 도면 내의 모든 해칭 객체(Hatch) 를 인식합니다.  
3\) 해칭의 표시 순서가 자동으로 변경되어, 다른 모든 객체 뒤에 배치됩니다.

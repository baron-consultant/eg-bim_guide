- 도면에서 선택한 객체를 임시로 숨기는 기능입니다.

![][image1]

## **1\. 객체 숨기기** [**시연 영상**](https://youtu.be/IsapFetzsyg)

1\) “HIDEOBJECTS” 명령어를 실행합니다.  
2\) 숨길 객체(선, 폴리선, 블록, 이미지, 해치 등)를 선택합니다.  
3\) ENTER 또는 SPACE BAR를 눌러 선택을 완료합니다.  
4\) 선택한 객체가 화면에서 숨겨져 보이지 않게 됩니다.  
 참고 : UNISOLATEOBJECTS 명령어를 실행하여 숨겨진 객체들을 다시 표시합니다.  
![][image2]

- 도면에 삽입된 블록의 위치를 화면에 표시하거나 선택하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/4QDOaXnoeTw)

1\) 명령창에 “FINDBLOCK” 입력합니다.  
2\) 블록 찾기 대화상자가 나타납니다.  
![][image2]  
3\) 검색할 블록 이름을 입력하거나, 목록에서 선택합니다.  
4\) 도면 내 해당 블록의 위치가 표시되며, 선택 또는 확대 이동됩니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/rl7-iTtnVtw)

1\) 명령창에 “FINDRADIUS” 입력합니다.  
2\) 명령창 안내에 따라 4개의 점을 순서대로 클릭합니다.

- 시작점(Start point) : 원호의 시작 지점
- 중간점(Mid point) : 원호의 중간 지점
- 끝점(End point) : 원호의 끝 지점
- 중심선이 지나갈 점(Center direction point) : 원의 중심이 존재하는 방향의 점  
  (즉, 원이 위로 휘어졌다면 위쪽, 아래로 휘어졌다면 아래쪽을 클릭)

3\) 명령창에 계산 결과가 출력됩니다.

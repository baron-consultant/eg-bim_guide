- 선택된 두 객체의 교차점을 지정한 반지름(Radius)으로 둥글게 연결해 주는 명령어 입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/hrUtQLmhU30)

- “FILLET” 명령어를 실행합니다.
- 첫 번째 객체, 두 번째 객체를 선택합니다.
- 또는 다음 옵션 중 하나를 선택합니다.

* Polyline : 선택한 폴리라인의 모든 모서리가 지정한 반지름으로 둥글게 변합니다.
* Radius : 반지름을 설정합니다.
* Trim : 기본 Trim으로 설정되어 있습니다.  
  &nbsp;- Trim : 원래 선분이 잘려나가고, 필렛 곡선만 남습니다.  
  &nbsp;- No trim : 원래 선분은 유지되고, 필렛 곡선이 추가됩니다.
* Settings : 설정창을 팝업합니다.
* Multiple : 설정된 반지름에 따라 여러 쌍의 객체에 연속적으로 필렛 적용이 가능합니다.

- 도면 내 삽입된 필드 객체를 표시할 때, 일반 텍스트 대신 회색 배경을 표시할지 여부를 제어하는 기능입니다.

![][image1]

## **1\. 필드 생성하기** [**필드 배경 표시 설정 시연 영상**](https://youtu.be/mVzcT5TtTcQ)

1\) “FIELDDISPLAY” 명령어를 실행합니다.  
2\) 명령창에서 ON / OFF 옵션을 선택합니다.

- ON : 필드가 회색 배경으로 표시됨 → 필드 객체를 쉽게 구분할 수 있음
- OFF : 필드가 일반 텍스트처럼 표시됨 → 출력 시처럼 배경 없이 표시됨

3\) 설정을 완료하면 도면에 있는 모든 필드 객체가 해당 방식으로 표시됩니다.  
4\) 확인(OK)을 누르면 도면에 MTEXT 형태로 필드가 삽입됩니다.  
![][image2]

- 도면에 삽입된 필드의 값을 자동으로 업데이트할 타이밍을 설정하는 기능입니다.

![][image1]

## **1\. 필드 갱신 조건 설정하기** [**필드 수동 업데이트 시연 영상**](https://youtu.be/P6mBgFzY_5k)

1\) “FIELDEVAL” 명령어를 실행합니다.  
2\) 명령창에서 0\~6 사이의 값을 입력합니다.

- 0 : 자동 갱신 안 함
- 1 : 도면 열기(Open) 시 갱신
- 2 : 도면 저장(Save) 시 갱신
- 4 : 플롯(Plot/출력) 시 갱신
- 8 : ETRANSMIT 실행 시 갱신
- 16 : 도면 재생성(REGEN) 시 갱신
- 31 : 위 조건 모두 적용 (1+2+4+8+16)

3\) 입력 후 Enter를 누르면 설정이 적용됩니다.

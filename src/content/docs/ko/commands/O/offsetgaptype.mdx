- 폴리라인을 간격띄우기 (OFFSET) 할 때 객체 사이의 모서리 처리 방법 변경

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/AwoxGEMhg7I)

1\) 명령창에 “OFFSETGAPTYPE” 명령어를 실행합니다.  
2\) 원하는 값 을 입력합니다 (0 \- Default, 1 \- Fillet, 2 \- Chamfer)  
3\) ENTER를 눌러 OFFSETGAPTYPE 변수 값을 변경합니다.  
4\) 이후 OFFSET 명령어를 실행하면 변경된 모서리의 처리 방법이 적용됩니다.  
![][image2]  
 0 \- Default 1 \- Fillet 2 \- Chamfer

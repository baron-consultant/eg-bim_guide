- 폴리선으로 변환하고 편집하는 명령어입니다.
- 폴리선으로 변환 가능한 객체는 원, 호, 타원, 3D 면, 선, 솔리드, 스플라인입니다.
- 3D폴리선으로 변환 가능한 객체는 선, 페이스입니다.
- 명령어를 실행하고 설정(S) 키워드를 입력하면 명령어 옵션 팝업창이 열립니다. 이 팝업창에서 명령어의 동작 옵션을 설정할 수 있습니다.
- “초기화” 버튼을 누르면 명령어 옵션이 기본 옵션으로 초기화됩니다.

## **1\. 폴리선으로 변환 후, 원본 삭제하기** [**시연 영상**](https://youtu.be/SAwHrLv3RUQ)

1\) “OTPLINE” 명령어를 실행하여 S 키워드를 입력합니다.  
2\) 폴리선 형식을 체크, 설정에서 “변환 후, 원본 삭제”를 체크하고 “확인” 버튼을 누릅니다.  
3\) 폴리선으로 변환할 객체를 선택하고 ENTER를 누릅니다.

## **2\. 폴리선으로 변환 후, 레이어 변경하기**

[**시연 영상**](https://youtu.be/EqqjFFjkv78)

1\) “OTPLINE” 명령어를 실행하여 S 키워드를 입력합니다.  
2\) 폴리선 형식을 체크, 설정에서 “변환 후, 레이어 변경”를 체크하고 “확인” 버튼을 누릅니다.  
3\) 폴리선으로 변환할 객체를 선택하고 ENTER를 누릅니다.

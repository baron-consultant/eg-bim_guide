- Offset 옵션 키워드

* Through: Offset 거리를 지정하는 대신 화면 상에 마우스로 점을 지정해 해당 점을 포함하거나 직선을 연장했을 때 지날 수 있는 객체를 생성합니다.
* Erase: 원본 객체를 삭제할지 여부를 설정합니다.
* Layer: Offset 명령어로 생성된 객체의 레이어가 원본 객체를 따를지 또는 현재 레이어가 될지 설정합니다.

![][image6]

- Offset 방향 옵션 키워드

* 화면 상에 마우스로 Offset이 실행된 객체가 생성될 방향을 지정합니다.
* Both sides: 원본 객체의 양방향에 Offset이 실행된 객체를 생성합니다.

## **1\. 간격을 띄우는 방법** [**시연 영상**](https://youtu.be/xLq1YUnQJTo)

1\) “OFFSET” 명령어를 실행합니다.  
2\) Offset 거리를 지정합니다.  
3\) Offset을 실행할 객체를 선택합니다.  
4\) Offset이 실행될 방향을 지정하고나면 원본 객체에서 Offset 거리만큼 띄워진 객체가 생성됩니다.

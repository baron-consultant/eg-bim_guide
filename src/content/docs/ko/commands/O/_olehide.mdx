* 도면 내에 삽입된 OLE 개체의 표시 여부를 제어하는 기능입니다.

![][image1]

## **1\. OLE 개체 숨기기 / 표시하기** [**시연 영상**](https://youtu.be/q2sl-Yz5JkE)

1\) 명령창에 **“OLEHIDE”** 를 입력합니다.  
2\) 아래 중 하나의 값을 선택하거나 입력합니다.

| 입력값 | 설명 |
| :---: | ----- |
| 0 | OLE 개체를 표시(On) 상태로 유지합니다. (기본값) |
| 1 | OLE 개체를 숨김(Hide) 상태로 설정합니다. |

3\) 엔터 키를 누릅니다.  
    참고 : “REGEN” 명령어를 사용하여 화면을 새로고침 합니다.

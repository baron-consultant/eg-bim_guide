title: ZOOMO
---

[image1]: ../../../../../assets/images/commands/zoomo/image1.png

- 선택한 객체(Object) 를 기준으로 그 객체가 화면에 최대한 크게 표시되도록 확대하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상 보기**]()

1\) 명령창에 “ZOOMO” 명령어를 입력합니다.  
2\) ENTER 키를 누릅니다.  
3\) 화면에서 확대하고 싶은 객체를 선택합니다.  
4\) ENTER 키를 누르면, 선택한 객체가 화면에 꽉 차도록 확대되어 표시됩니다.

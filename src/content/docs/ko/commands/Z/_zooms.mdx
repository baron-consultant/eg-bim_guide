- 화면의 표시 배율을 사용자가 입력한 배율 값으로 조정하는 기능입니다.

![][image1]

## **1\. 배율로 화면 확대·축소하기** [**시연 영상 보기**]()

1\) 명령창에 “ZOOMS” 명령어를 입력합니다.  
2\) ENTER 키를 누릅니다.  
3\) 원하는 배율 값 또는 형식을 입력합니다.  
4\) 입력한 배율 값에 따라 화면이 확대 또는 축소됩니다.  
![][image2]

- 지정한 점을 화면의 중심으로 설정하고, 입력한 폭 값에 따라 화면을 확대 또는 축소하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상 보기**]()

1\) 명령창에 “ZOOMC” 명령어를 입력합니다.  
2\) ENTER 키를 누릅니다.  
3\) 화면에서 중심으로 하고 싶은 점을 클릭합니다.  
4\) 화면에 표시할 폭을 숫자로 입력하거나 2X, 0.5X 등의 배율 형식으로 입력합니다.  
5\) ENTER 키를 누르면 지정한 중심점을 기준으로 확대/축소가 적용됩니다.  
![][image2]

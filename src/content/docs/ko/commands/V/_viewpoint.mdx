* 도면에서 시점을 지정하여, 화면을 특정 방향(X, Y, Z축 기준)으로 바라보는 3차원 뷰를 설정하는 명령어입니다.

![][image1]

## **1\. 3D 시점 지정하기** [**시연 영상**]()

1\) 명령창에 “VIEWPOINT” 을 입력합니다.  
2\) 시점을 좌표로 직접 입력하거나, 회전 옵션을 사용합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Perspective | 원근 뷰를 켜거나 끔. 입체감을 표현할 때 사용합니다. |
| PLAN | 현재 UCS 기준으로 평면(위에서 본) 뷰로 전환합니다. |
| Rotate | 시점을 회전시켜 모델을 다른 각도에서 관찰합니다. |

3\) 좌표 또는 옵션 입력 후 엔터 키를 누릅니다.  
4\) 화면이 지정한 3D 시점으로 전환됩니다.

* 페이퍼 공간에서 뷰포트의 표시 영역을 잘라내거나 특정 형태(사각형, 다각형 등)로 경계선을 지정하는 명령어입니다.

![][image1]

## **1\. 뷰포트 클리핑 설정하기** [**시연 영상**]()

1\) 레이아웃 탭으로 전환합니다.  
2\) 명령창에 “VPCLIP” 을 입력합니다.  
3\) 클리핑할 뷰포트를 선택합니다.  
4\) 클리핑 옵션을 지정합니다.

| 옵션 | 설명 |
| :---: | ----- |
| Polygonal | 새로 다각형 형태의 경계를 지정합니다. |
| Delete | 기존 클리핑 경계를 삭제하여 원래 사각형 뷰포트로 복원합니다. |

5\) 경계를 마우스로 지정하면 뷰포트가 해당 형태로 잘립니다.

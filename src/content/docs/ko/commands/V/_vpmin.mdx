- 현재 최대화되어 있는 뷰포트를 원래의 도면 공간 크기로 되돌리는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) Layout 탭으로 이동한다.  
2\) “VPMAX” 명령어로 뷰포트를 최대화 합니다.  
2\) 명령창에 “VPMIN” 를 입력합니다.  
3\) ENTER 키를 누릅니다.  
4\) 확대되어 있던 뷰포트가 원래 Layout 공간 크기로 복구됩니다.

- 도면에 삽입된 래스터 이미지의 테두리 표시 여부와 플롯 출력 여부를 제어하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/kIuLQPABawU)

1\) 명령창에 “IMAGEFRAME” 명령어를 입력합니다.  
2\) 옵션으로 0, 1, 2 중 하나의 값을 입력한다.

- 0 : 프레임 표시 안 함 (화면과 출력 모두 프레임 없음 → 이미지 편집 불가)
- 1 : 프레임 표시 및 출력 (화면에도 보이고, 출력 시에도 프레임 출력됨)
- 2 : 프레임 표시만 함 (화면에서만 프레임 표시, 출력 시에는 프레임 출력 안 됨)

3\) ENTER를 눌러 설정을 완료한다.

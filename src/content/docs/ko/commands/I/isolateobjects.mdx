- 도면에서 선택한 객체만 화면에 표시하고, 나머지 모든 객체는 임시로 숨기는 기능입니다.

![][image1]

## **1\. 숨겨진 객체 복원하기** [**선택 객체 제외하고 숨기기 시연 영상**](https://youtu.be/i5mXEyJ9hPI)

1\) “ISOLATEOBJECTS” 명령어를 실행합니다.  
2\) 화면에 표시할 객체(선, 폴리선, 블록, 이미지, 해치 등)를 선택합니다.  
3\) ENTER 또는 SPACE BAR를 눌러 선택을 완료합니다.  
4\) 선택한 객체만 화면에 표시되고, 다른 객체는 전부 숨겨집니다.  
 참고 : UNISOLATEOBJECTS 명령어를 실행하여 숨겨진 객체들을 다시 표시합니다.  
![][image2]

- 엑셀에서 좌표값을 읽어 객체를 그립니다.

## **1\. 엑셀에서 좌표값을 읽어 객체 그리기** [**시연 영상**](https://youtu.be/dlEJJwDrZeU)

1. 명령어를 실행하면 명령어 옵션 팝업창이 열립니다.

- 팝업창에서 “확인” 버튼을 누르면 옵션이 저장되고 엑셀의 영역선택 기능이 활성화 됩니다.

![][image1]

- 엑셀에서 가져올 대상에 맞는 데이터 선택 후 확인 버튼 클릭 시 설정에 따라 객체가 삽입됩니다.  
  ![][image2]

**옵션 설명**

- “가져올 대상” 에서 옵션션을 설정하여 데이터 형식을 지정할 수 있습니다.
- 좌표 연결선을 설정하여 좌표를 연결하는 선 객체 설정 할 수 있습니다.
- 정점 객체 삽입 항목을 설정하여 좌표에 따른 정점에 점, 원, 블록 객체를 삽입하거나 ON/OFF를 설정할 수 있습니다.

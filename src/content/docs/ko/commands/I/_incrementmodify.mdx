title: INCREMENTMODIFY
---

[image1]: ../../../../../assets/images/commands/incrementmodify/image1.png

* 도면 내에서 선택한 객체의 Z 값을 지정한 시작값과 증분값 만큼 순차적으로 증가시키는 명령어입니다.

## **1\. Z값 연속 증가하기** [**시연 영상**]()

1\) 명령창에 “INCREMENTMODIFY” 를 입력합니다.  
2\) 설정 창에서 필요한 옵션을 설정합니다.  
![][image1]  
3\) 도면에서 Z 값을 변경할 객체를 선택합니다.  
4\) 엔터 키를 누르면, 선택한 객체들의 Z 값이 순차적으로 증가합니다.  
  참고 : “객체 선택 → 엔터 → 다시 객체 선택 → 엔터” 과정을 반복할 때마다 Z 값이 누적 증가합니다.

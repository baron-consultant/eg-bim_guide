- 도면에 삽입된 래스터 이미지(JPEG, PNG 등)의 화질 품질 수준을 제어하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/ezq317_5Cmk)

1\) 명령창에 “IMAGEQUALITY” 명령어를 입력합니다.  
2\) 옵션으로 High 또는 Draft 값을 입력한다.

- High : 높은 화질로 표시 → 화질(선명도) 우선
- Draft : 낮은 화질로 표시 → 성능(속도) 우선

![][image2]

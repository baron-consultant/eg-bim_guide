* 도면에 삽입된 PDF 언더레이 객체의 프레임(테두리선) 표시 여부를 제어하는 명령어입니다.

![][image1]

## **1\. PDF 프레임 표시 설정하기** [**시연 영상**]()

1\) 명령창에 “PDFFRAME” 을 입력합니다.  
2\) 프레임 표시 상태를 지정하는 값을 입력합니다.

| 값 | 설명 |
| :---: | ----- |
| 0 | 프레임을 표시하지 않음, 인쇄 시에도 나타나지 않습니다. |
| 1 | 프레임을 표시함, 인쇄 시에도 함께 인쇄됩니다. |
| 2 | 프레임을 표시함, 인쇄 시에는 표시되지 않습니다. |

3\) 엔터 키를 눌러 설정을 적용합니다.  
    참고 :  값 변경 후 화면에 즉시 반영되지 않을 경우, "REGEN" 명령으로 화면을 새로 고침합니다.

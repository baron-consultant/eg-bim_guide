* 도면 공간에서 선종류의 축척을 뷰포트 배율에 맞춰 조정할지 여부를 설정하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “PSLTSCALE” 을 입력합니다.  
2\) 다음 중 하나의 값을 입력합니다.

- ON : 뷰포트 배율 무시 \- 각 뷰포트의 실제 선종류 간격(LTSCALE)을 그대로 사용  
- OFF : 뷰포트 배율 반영 \- 모든 뷰포트에서 동일한 선 패턴 간격으로 표시 (기본값)

3\) 엔터 키를 누릅니다.  
4\) 명령창에 “REGEN” 하여  화면을 새로 고침 합니다.

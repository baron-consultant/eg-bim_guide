- 현재 작업 중인 모델 공간(작업용 공간) 에서 도면 공간(출력용 공간) 으로 전환하는 명령입니다.

![][image1]

## **1\. 도면 공간으로 전환하기** [**시연 영상**](https://youtu.be/p2oe01KizMU)

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “PSPACE” 명령어를 입력합니다.  
3\) ENTER 키를 누릅니다.  
4\) 현재 작업 공간이 도면 공간 으로 전환됩니다.  
 참고 : 도면 공간에서 모델 공간으로 돌아가려면 “MSPACE” 명령을 사용합니다.

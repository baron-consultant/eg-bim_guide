* 현재 도면을 인쇄하거나 PDF 등으로 출력하는 기능입니다.

![][image1]

## **1\. 인쇄하기** [**시연 영상**]()

1\) 명령창에 “RESCALEWITHDIM” 을 입력합니다.  
2\) Plot 대화상자가 나타납니다.  
![][image2]  
3\) 필요한 항목을 설정합니다.

- Printer/Plotter : 출력 장치 선택 (예: DWG To PDF, HP DesignJet 등)  
- Paper Size : 용지 크기 선택 (예: A4, A3, A1 등)  
- Plot Area : 출력할 영역 지정  
- Plot Style : 선 굵기 및 색상 스타일 지정

4\) 미리보기를 클릭하여 인쇄 결과를 확인합니다.  
5\) 설정이 완료되면 인쇄하기 버튼을 눌러 인쇄를 실행합니다.

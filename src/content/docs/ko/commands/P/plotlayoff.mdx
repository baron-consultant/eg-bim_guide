- 도면에서 특정 레이어를 화면에서는 표시하되, 출력 에서는 제외하도록 설정하는 기능입니다.

![][image1]

## **1\. 레이어 출력 제외하기** [**시연 영상**](https://youtu.be/EQI5AHv0H6I)

1\) 명령창에 “PLOTLAYOFF” 명령어를 입력합니다.  
2\) 출력에서 제외할 객체(레이어에 속한 객체 중 하나) 를 선택합니다.  
3\) 해당 객체가 속한 레이어가 플롯 제외 상태로 변경됩니다.  
4\) 이후 “PLOT” 명령어를 실행하면 해당 레이어의 객체들은 출력되지 않습니다.  
![][image2]

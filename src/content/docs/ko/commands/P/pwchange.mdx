- 선택한 객체의 선 두께를 일괄적으로 변경합니다.

![][image1]

## **1\. 객체 선 두께 변경** [**시연 영상**](https://youtu.be/YuyEVKOXsAo)

1\) “PWC” 또는 “PWCHANGE” 명령어를 실행합니다.  
2\) 선 두께를 변경할 객체(선, 폴리선, 원, 사각형 등)을 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 원하는 선 두께 값(Lineweight) 을 입력합니다.  
5\) ENTER 또는 SPACE BAR를 누릅니다.  
6\) 선택한 객체의 선 두께가 입력한 값으로 일괄 변경됩니다.

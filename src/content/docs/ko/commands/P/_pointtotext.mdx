- 도면에 배치된 점 객체의 좌표값을 자동으로 읽어 해당 위치에 좌표 텍스트를 생성하는 명령어입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/5MxNN6c_wbc)

1\) 명령창에 “POINTTOTEXT” 를 입력합니다.  
2\) 대화상자에서 필요한 항목을 설정합니다.  
![][image2]  
2\) 좌표를 표시할 점 객체 선택합니다.  
3\) 명령이 자동으로 각 점의 좌표(X, Y, Z)를 계산합니다.  
4\) 계산된 좌표값을 기준으로 각 점 근처에 텍스트 객체가 생성됩니다.

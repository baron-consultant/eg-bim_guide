- 도면(Layout)을 생성하거나 삭제하고, 기존 레이아웃을 이름 변경 또는 활성화하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) 명령창에 “PAGESETUP” 입력합니다.  
2\) 페이지 설정 관리자 창이 나타납니다.  
![][image2]  
3\) 새로 만들기 또는 수정 버튼을 선택합니다.  
4\) 인쇄 환경(프린터, 용지 크기, 플롯 영역, 배율, 스타일 테이블 등)을 설정합니다.  
![][image3]  
5\) 확인을 눌러 설정을 저장합니다.  
6\) 이후 해당 설정을 Layout 탭이나 Plot 창에서 불러와 적용할 수 있습니다.

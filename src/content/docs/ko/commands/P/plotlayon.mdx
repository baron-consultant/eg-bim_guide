- “PLOTLAYOFF” 명령어로 출력이 비활성화된 레이어를 다시 출력 가능 상태으로 되돌리는 기능입니다.

![][image1]

## **1\. 레이어 출력 복권하기** [**시연 영상**](https://youtu.be/xbZi8UKc1VE)

1\) 명령창에 “PLOTLAYON” 명령어를 입력합니다.  
2\) 출력에서 포함할 객체(레이어에 속한 객체 중 하나) 를 선택합니다.  
3\) “PLOTLAYOFF” 명령어로 출력이 꺼져 있던 객체가 속한 레이어가 플롯 포함 상태로 변경됩니다.
4\) 이후 “PLOT” 명령어를 실행하면 해당 레이어의 객체들은 출력됩니다.
![][image2]

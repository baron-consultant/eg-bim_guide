title: PLINECONVERTMODE
---

[image1]: ../../../../../assets/images/commands/plineconvertmode/image1.png

* 스플라인, 호, 선 등을 폴리선으로 변환할 때, 폴리선의 세그먼트 형식을 결정합니다.

![][image1]

## **1\. 폴리선 변환모드 설정** [**시연 영상**]()

1\) 명령창에 “PLINECONVERTMODE” 를 입력합니다.  
2\) 필요한 옵션을 선택합니다.

| 값 | 동작 | 용도 |
| :---: | :---: | ----- |
| 0 | 직선 세그먼트로 변환 | 단순한 다각형 형태로 필요할 때 사용 |
| 1 | 호 세그먼트로 변환 | 곡률을 더 잘 보존, 매끄러운 결과 필요할 때 사용 |

3\) 이후 스플라인을 폴리선으로 변환하면 해당 모드가 반영됩니다.

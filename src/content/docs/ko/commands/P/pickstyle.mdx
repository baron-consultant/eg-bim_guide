[image1]: ../../../../../assets/images/commands/pickstyle/image1.png

- 도면에서 그룹 및 연관 해치를 객체처럼 선택할 수 있는지 여부를 제어하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/zeKjQhPC5Ac)

1\) 명령창에 “PICKSTYLE” 명령어를 입력합니다.  
2\) 원하는 값을 입력합니다.

- 0 : 그룹 및 연관 해치를 무시 → 개별 객체만 선택됨
- 1 : 그룹을 하나의 객체처럼 선택 가능
- 2 : 연관 해치를 하나의 객체처럼 선택 가능
- 3 : 그룹과 연관 해치 모두 하나의 객체처럼 선택 가능

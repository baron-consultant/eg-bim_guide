title: PLINEGEN

[image1]: ../../../../../assets/images/commands/plinegen/image1.png

- 폴리선의 정점마다 선 스타일을 적용할지 여부를 설정하는 명령어입니다.
- 설정 ON \=\> PLINE 명령어로 그려지는 폴리선의 정점마다 현재 선 스타일이 적용됩니다,  
  설정 OFF \=\> PLINE 명령어로 그려지는 폴리선의 시작점에서만 현재 선 스타일이 시작되고 정점에서 선 스타일이 끊기지 않습니다, (연속 적용)

![][image1]

## **1\. 설정을 켜고 끄는 방법** [**시연 영상**](https://youtu.be/wL3qiCXVaek)

1\) “PLINEGEN” 명령어를 실행합니다.  
2\) ON 하고 싶은 경우 명령어 창에 ON 또는 1을 입력하고,  
OFF 하고 싶은 경우 명령어 창에 OFF 또는 0을 입력합니다.

- 폴리선을 생성합니다.
- 여러가지 옵션을 이용하여 폴리선을 생성하거나 객체 타입이 폴리선을 그릴 수 있습니다.

![][image1]

- 폴리선의 시작점 지정 옵션 키워드

* Follow 또는 ENTER: last point를 폴리선의 시작점으로 지정
* 화면 상에 마우스로 지정

![][image2]

- 폴리선의 Next point 지정 옵션 키워드

— 시작점 지정 이후부터

- Arc: 객체 타입이 폴리선인 호를 생성합니다.
- Distance: 길이, 각도를 입력하여 폴리선을 생성합니다.
- Follow: 직전에 그려졌던 폴리선과 각도가 일치하는 폴리선을 생성합니다.
- Halfwidth: 그려질 폴리선의 시작 너비의 절반 값과 끝 너비의 절반 값을 설정합니다.
- Width: 그려질 폴리선의 시작 너비와 끝 너비를 설정합니다.

— 2번째 포인트 지정 이후부터

- Undo: 현재 폴리선 생성 작업을 취소하고 직전에 진행되었던 폴리선 생성 작업으로 되돌아갑니다.

— 3번째 포인트 지정 이후부터

- Close: 닫힌 형태의 폴리선의 생성합니다.

![][image3]

- 객체 타입이 폴리선인 호 그리기 옵션 키워드

* Angle: Included angle for segment를 입력하여 호를 그립니다.
* Center: 화면 상에 마우스로 호의 중심점을 지정하여 호를 그립니다.
* Close: 폴리선의 시작점을 끝점으로 가지는 호가 생성되어 폴리선이 닫히고 명령어가 종료됩니다.
* Direction: 화면 상에 마우스로 Direction from the start를 지정하고 호를 그립니다.
* Halfwidth: 호의 선 너비의 절반 값을 설정합니다.
* Line: 직선 그리기 옵션으로 변경합니다.
* Radius: 호의 반지름을 설정합니다.
* Secondpt: 호가 포함할 Second point를 화면 상에 마우스로 지정하고 호를 그립니다.
* Width: 호의 선 너비를 설정합니다.
* Undo: 현재 생성 작업을 취소하고 직전에 진행되었던 생성 작업으로 되돌아갑니다.

## **1\. 폴리선을 그리는 방법** [**시연 영상**](https://youtu.be/BAW0OXStXp8)

1\) “PLINE” 명령어를 실행합니다.  
2\) 폴리선의 시작점, Next Point를 지정합니다.  
3\) ENTER 또는 ESC 키를 누르면 폴리선이 생성되고 명령어가 종료됩니다.

## **2\. 객체 타입이 폴리선인 호를 그리는 방법** [**시연 영상**](https://youtu.be/Vvz5_fQFzPI)

1\) “PLINE” 명령어를 실행합니다.  
2\) 폴리선의 시작점을 지정합니다.  
3\) Arc 키워드를 입력합니다.  
4\) 호의 속성 설정이 따로 필요한 경우 키워드를 선택하고= 값을 입력합니다.  
5\) End of arc를 지정하면 호가 생성됩니다.  
6\) ENTER 또는 ESC 키를 누르면 명령어가 종료됩니다.

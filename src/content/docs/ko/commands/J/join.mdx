- 떨어져 있는 선 을 연결시키거나 이어저 있는 여러개의 선을 하나로 결합하는 명령어 입니다.

![][image1]

## **1\. 선 연결하기** [**시연 영상**](https://youtu.be/xJXm7yigqLQ)

1\) 명령창에 “JOIN” 명령어를 입력합니다.  
2\) 연결하려는 두 개 이상의 선을 선택 후 ENTER를 누릅니다.

- 연결은 LINE \+ LINE 객체일때만 가능합니다.
- 연결 시 동일 선상에 (X축 or Y축) 있을때만 가능합니다.

3\) 여러개의 선이 1개의 선으로 연결됩니다.  
![][image2]![][image3]

## **2\. 선 결합하기** [**시연 영상**](https://youtu.be/OEFY9IeBu_U)

1\) 명령창에 “JOIN” 명령어를 입력합니다.  
2\) 결합하려는 두 개 이상의 선 을 선택 후 ENTER를 누릅니다.

- 결합은 선의 종류와 상관 없이 결합됩니다.
- 결합 시 각 선의 끝이 붙어 있을때만 가능합니다.

3\) 여러개의 선이 1개의 선(폴리라인) 으로 결합됩니다.

- 점프선을 그립니다.
- 명령어를 실행하고 설정(S) 키워드를 입력하면 명령어 옵션 팝업창이 열립니다. 이 팝업창에서 명령어의 동작 옵션을 설정할 수 있습니다.
- “초기화” 버튼을 누르면 명령어 옵션이 기본 옵션으로 초기화됩니다.

![][image1]

- 점프선을 그리기전에 옵션값을 팝업창에서 설정할 수 있습니다.

![][image2]

## **1\. 기준선과 교차하는 선 선택해 점프선 그리기** [**시연 영상**](https://youtu.be/HxDVxvk62qM)

1\) “JMP” 명령어를 실행합니다.  
2\) 설정(S) 키워드를 선택/입력합니다.  
3\) 명령어 옵션 팝업창의 기본 설정에서 “기준선과 교차하는 선 선택해 그리기”를 클릭합니다.  
4\) “확인” 버튼을 누르면 설정이 저장됩니다.  
5\) 점프선을 그릴 기준 선을 선택합니다.(선, 호, 폴리선, 스플라인만 선택이 가능합니다.)  
6\) 점프선을 참고할 객체들을 선택합니다.(옵션값의 교차선 필터를 기준)  
7\) 볼록 방향을 설정합니다.  
8\) ESC키로 명령어를 종료합니다.

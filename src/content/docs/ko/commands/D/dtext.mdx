## **1\. 단일 텍스트 작성하기** [**시연 영상**](https://youtu.be/FL-7sKf508M)

1\) “DTEXT” 또는 “DT” 명령어를 실행합니다.  
2\) 화면에 텍스트를 배치할 시작점을 지정합니다.  
3\) 텍스트의 높이(문자 크기)를 입력합니다.  
4\) 텍스트의 회전 각도를 입력합니다.  
5\) 입력할 문자열을 키보드로 입력합니다.  
6\) ESC 키를 누르면 텍스트 입력이 완료되고 화면에 배치됩니다.  
![][image2]

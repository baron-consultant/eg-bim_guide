* 치수를 객체와의 연관에서 분리하는 기능입니다.

![][image1]

## **1.치수 연결 해제** [**시연 영상**]()

1\) 명령창에 “DIMDISASSOCIATE” 를 입력합니다.  
2\) 연결을 해제할 치수 객체를 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 선택한 치수는 더 이상 원본 객체의 이동/수정에 따라 자동 갱신되지 않습니다.

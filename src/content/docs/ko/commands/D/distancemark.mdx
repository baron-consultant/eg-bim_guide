* 도면 내에서 일정 간격으로 거리 마크(Distance Mark) 를 반복 표시하는 기능입니다.

![][image1]

## **1\. 거리 마크 표시하기** [**시연 영상**](https://youtu.be/BuHqiFDpbgM)

1\) 명령창에 “DISTANCEMARK” 을 입력합니다.  
2\) 설정 창에서 필요한 옵션을 지정합니다.  
![][image2]  
3\) 설정 후 **확인** 버튼을 클릭합니다.  
4\) 객체를 선택하고 시작점을 클릭합니다.  
5\) 설정된 반복 횟수만큼 거리선(Line) 과 거리값 텍스트(Text) 가 자동 생성됩니다.

title: DIMORDINATE
---

[image1]: ../../../../../assets/images/commands/dimordinate/image1.png

* 특정 점의 X좌표 또는 Y좌표 값을 치수로 표시하는 기능입니다.

![][image1]
 
## **1\. 좌표 표시하기** [**시연 영상**]()

1\) 명령창에 “DIMORDINATE” 를 입력합니다.  
2\) 좌표를 표시할 점을 클릭합니다.  
3\) 치수선의 위치를 클릭합니다.  
4\) 표시할 방향으로 마우스를 이동시키면 X 또는 Y 좌표값이 자동으로 나타납니다.

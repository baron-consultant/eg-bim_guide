![][image1]

## **1\. 중복 객체 삭제하기** [**시연 영상**](https://youtu.be/mWqAou2etiI)

1\) “DELETEDUPLICATEENTITIES” 명령어를 실행합니다.  
2\) 중복 삭제 옵션 대화상자가 나타납니다.  
![][image2]  
3\) 원하는 옵션을 설정 후 확인을 클릭합니다.  
4\) 중복 여부를 검사할 객체를 선택합니다.  
5\) ENTER 또는 SPACE BAR를 눌러 선택을 완료합니다.  
6\) 선택한 객체들에서 중복 요소가 제거됩니다.

* 객체와 치수 간의 연관 관계를 다시 연결하는 기능입니다.

![][image1]

## **1\. 치수 연결 복원하기** [**시연 영상**](https://youtu.be/EJ2-6wvEiJc)

1\) 명령창에 DIMREASSOCIATE 를 입력합니다.  
2\) 비연관형 상태의 치수를 선택합니다.  
3\) 첫번째, 두 번째 점을 각각을 지정합니다.  
4\) 엔터 키를 누르면, 해당 치수가 선택한 객체와 다시 연관되어 자동 갱신형 치수로 복원됩니다.

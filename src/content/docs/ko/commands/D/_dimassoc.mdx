* 치수 객체가 도면의 요소 와 어떤 방식으로 연관되어 있는지를 제어하는 시스템 변수 입니다.

![][image1]

## **1\. 치수 연결 설정** [**시연 영상**]()

1\) 명령창에 “DIMASSOC” 를 입력합니다.  
2\) 아래의 옵션 중 하나를 선택하여 입력합니다.

| 값 | 설명 |
| :---: | ----- |
| 0 | 치수 요소(선, 화살표, 텍스트 등)가 개별 객체로 분리되어, 연관되지 않은 상태로 생성됩니다. |
| 1 | 치수가 비연관형으로 생성됩니다. 치수 객체는 단일 객체지만, 원본 요소(선, 점 등)과 연결되지 않습니다. |
| 2 | 치수가 연관형으로 생성됩니다. 치수는 참조한 객체와 연결되어 있으며, 객체가 이동, 수정되면 치수 값도 자동 갱신됩니다. (기본 권장값) |

3\) 엔터 키를 누르면 옵션이 적용됩니다.

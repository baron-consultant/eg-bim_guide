title: DIMLINEPOSITION

[image1]: ../../../../../assets/images/commands/dimlineposition/image1.png

* 선택한 치수선 들을 정렬하여 일정한 기준선 위로 위치를 맞추는 기능입니다.

![][image1]

## **1\. 치수선 정렬하기** [**시연 영상**]()

1\) 명령창에 **“**DIMLINEPOSITION**”** 을 입력합니다.  
2\) 정렬할 치수선을 여러 개 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 정렬 기준점을 클릭합니다.  
5\) 선택한 모든 치수선이 해당 기준선 위치로 정렬됩니다.

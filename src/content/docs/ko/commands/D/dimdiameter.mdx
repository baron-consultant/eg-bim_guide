- 선택한 원 또는 호의 지름을 측정하여 치수 문자로 표시합니다.

## **1\. 지름 치수 입력 방법** [**시연 영상**](https://youtu.be/YG4XG2NXVzs)

1\) DIMDIAMETER 명령을 실행합니다.  
![][image1]  
2\) 도면에 지름을 측정할 원 또는 호를 선택합니다.  
![][image2]  
3\) 치수 문자를 원하는 위치에 삽입합니다.  
![][image3]  
![][image4]

## **2\. 지름 치수 문자의 각도 변경** [**시연 영상**](https://youtu.be/4h1Fu4pU9lc)

1\) DIMDIAMETER 명령을 실행하고, 도면에 지름을 측정할 원 또는 호를 선택합니다.  
2\) Angle 옵션을 클릭 후 각도을 입력하면, 입력한 각도로 치수 문자가 기입 됩니다. (기본값은 0도)  
![][image5]  
![][image6]

## **3\. 지름 치수 문자를 여러줄 문자로 입력하는 방법** [**시연 영상**](https://youtu.be/ECs3SXsBAeQ)

1\) DIMDIAMETER 명령을 실행하고, 도면에 지름을 측정할 원 또는 호를 선택합니다.  
2\) Mtext 옵션을 클릭하여 Text Editor 툴바가 나타나고 치수문자가 편집 상태가 되면, 여러줄로 문자를 입력합니다.  
![][image7]  
![][image8]  
3\) Text Editor 툴바의 Done버튼(V)을 클릭하거나, 도면의 여백 부분을 클릭하여 편집을 종료합니다.  
![][image9]

## **4\. 지름 치수 문자를 변경하는 방법** [**시연 영상**](https://youtu.be/sS0_D01lDKQ)

1\) DIMDIAMETER 명령을 실행하고, 도면에 지름을 측정할 원 또는 호를 선택합니다.  
2\) Text 옵션을 클릭하고, 변경할 문자열을 입력합니다.  
![][image10]  
3\) ENTER를 누르고, 도면에서 지름 치수를 원하는 위치에 기입합니다.  
![][image11]

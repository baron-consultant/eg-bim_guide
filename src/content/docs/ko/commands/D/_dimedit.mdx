* 치수의 텍스트 위치, 회전 각도, 또는 내용 수정 등을 빠르게 편집하는 기능입니다.

![][image1]

## **1\. 치수 위치 편집** [**시연 영상**]()

1\) 명령창에 “DIMEDIT” 를 입력합니다.  
2\) 아래의 옵션 중 하나를 선택하여 입력합니다.

| 값 | 설명 |
| :---: | ----- |
| Home(H) | 치수 텍스트를 원래 위치(기준 위치)로 되돌립니다. |
| New (N) | 기존 치수 값을 새 텍스트로 교체합니다. |
| Rotate (R) | 치수 텍스트를 수평 또는 수직 방향으로 회전시킵니다. |
| Oblique (O) | 치수 보조선의 기울기(Oblique Angle)를 지정하여 비스듬히 표시합니다. |

3\) 엔터 키를 누르면 옵션이 적용됩니다.

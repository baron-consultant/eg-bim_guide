* 여러 개의 분리된 치수를 하나의 연속된 치수로 결합하는 기능입니다.

![][image1]

## **1\. 치수 결합하기** [**시연 영상**](https://youtu.be/Ax2W64MrRms)

1\) 명령창에 “DIMJOIN” 를 입력합니다.  
2\) 기준이 될 첫 번째 치수 객체를 선택합니다.  
3\) 결합할 나머지 치수 객체들을 순서대로 선택합니다.  
4\) 엔터 키를 누릅니다.  
5\) 선택한 치수들이 자동으로 정렬되어 하나의 연속 치수선으로 결합됩니다.

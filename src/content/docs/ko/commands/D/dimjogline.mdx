- 선형치수(DIMLINEAR) 또는 정렬치수(DIMALIGNED)에 꺽기선을 추가하거나 삭제 합니다.

## **1\. 선형치수에 꺽기선 추가** [**시연 영상**](https://youtu.be/nnYI1bDZRX4)

1\) DIMJOGLINE 명령을 실행하고, Add를 선택합니다.  
![][image1]  
2\) 꺽기선을 추가할 치수를 선택하고, ENTER를 누릅니다.  
![][image2]  
3\) 꺽기선을 추가할 위치에서 마우스를 클릭합니다.  
![][image3]

## **2\. 선형치수에 꺽기선 삭제** [**시연 영상**](https://youtu.be/Xp8Ja1C1UxM)

1\) DIMJOGLINE 명령을 실행하고, Remove를 선택합니다.  
![][image4]  
2\) 꺽기선을 삭제할 선형치수를 선택합니다.  
![][image5]  
![][image6]

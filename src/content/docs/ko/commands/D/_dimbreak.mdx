* 치수선 또는 연장선이 다른 선, 객체와 겹칠 때 자동으로 간격을 만들어주는 명령어입니다.

![][image1]

## **1\. 치수선 분리** [**시연 영상**]()

1\) 명령창에 **“**DIMBREAK**”** 을 입력합니다.  
2\) 다음과 같은 순서로 명령창에 따라 값을 입력합니다.

- 기준 축척 값 입력 : 현재 도면의 기존 축척 값  
- 축척 비율 변경 값 입력 : 새로 적용할 축척 값

3\) 변경할 치수 객체를 선택합니다.  
4\) 엔터 키를 누르면 선택된 모든 치수 객체의 배율이 입력한 비율에 맞게 조정됩니다.

* 치수 텍스트의 위치나 방향을 수정하는 기능입니다.

![][image1]

## **1\. 텍스트 위치 수정하기** [**시연 영상**](https://youtu.be/-Qrk0du9UOU)

1\) 명령창에 “DIMTEDIT” 입력합니다.  
2\) 수정할 치수선을 클릭합니다.  
3\) 명령창에 다음과 같은 옵션이 나타납니다.  
4\) 필요한 옵션을 선택하거나 마우스로 직접 위치를 지정합니다.

| 옵션 | 설명 |
| :---: | :---: |
| Letf | 텍스트를 치수선의 왼쪽으로 정렬 |
| Right | 텍스트를 치수선의 오른쪽으로 정렬 |
| Center | 텍스트를 치수선의 중간으로 정렬 |
| Home | 텍스트를 치수선의 처음 위치로 정렬 |
| Angle | 텍스트를 지정한 각도로 회전 |

5\) 치수 위치가 수정됩니다.

- 선택한 객체가 속한 레이어만 화면에 표시(격리) 하고, 나머지 레이어는 꺼지거나 잠기도록 하여 선택 레이어만 쉽게 작업할 수 있도록 도와줍니다.

![][image1]

## **1\. 레이어 단독 표시하기** [**시연 영상**](https://youtu.be/N3kK31lIpEQ)

1\) 명령창에 “LAYISO” 명령어를 입력합니다.  
2\) 화면에서 격리하고 싶은 객체(또는 여러 개) 를 선택한다.  
3\) ENTER 키를 누른다.  
4\) 선택한 객체의 레이어만 화면에 표시되고, 다른 모든 레이어는 꺼지거나 잠긴 상태로 변경된다.  
 참고 : 격리된 상태를 해제하려면 “LAYUNISO” 명령어를 사용합니다.  
![][image2]

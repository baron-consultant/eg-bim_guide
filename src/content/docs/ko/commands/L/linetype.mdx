- 선 스타일 관리자 화면을 로드합니다.
- 선 스타일 관리자 화면에서는 새로운 선을 생성하거나, 기존의 선을 편집 또는 삭제할 수 있습니다.

  ![][image1]

- 선스타일 관리자는 좌측에 선 스타일 목록과 우측에 선 스타일 편집화면으로 구성되어 있습니다.
- 선스타일 관리자는 다양한 선꼴을 아래와 같은 기본 제공합니다.
  - 기본 DASH 형태의 스타일
    ![][image2]
  - 선 \+ 문자 형태의 스타일
    ![][image3]
  - 선 \+ 블록 형태의 스타일
    ![][image4]
  - 선 \+ 문자 \+ 블록 형태의 스타일
    ![][image5]

## **1\. 새로운 선 스타일을 생성하는 방법** [**시연 영상**](https://youtu.be/hCoHj3J3PFk)

1\) 좌측의 선 스타일 목록의 아래에 “새로 만들기” 를 클릭합니다.  
2\) 스타일 이름 및 스타일 상세 내용을 편집 후, 우측 상단의 “적용” 버튼을 클릭하면 새로운 선 스타일이 생성됩니다. “취소” 를 클릭하면, 진행중이던 생성 작업은 취소됩니다.

<br />

## **2\. 선 스타일을 삭제하는 방법** [**시연 영상**](https://youtu.be/xcONNqU2ehY)

1\) 삭제를 원하는 선을 마우스 오른쪽 버튼을 클릭하면, 삭제 메뉴가 생성되며 삭제를 누를 시 해당 선은 삭제됩니다. (단, 삭제가 불가능한 선은 삭제되지 않습니다.)
![][image6]

<br />

## **3\. 선 스타일을 편집하는 방법** [**시연 영상**](https://youtu.be/1XFnmWF8_AE)

1\) 편집을 원하는 스타일을 좌측 목록에서 선택 후, 오른쪽 상세 화면 우측 상단의 “수정” 버튼을 클릭합니다.  
2\) 편집 후, “적용” 버튼을 클릭하면 도면에 적용됩니다. “취소” 버튼을 클릭 시, 변경내용은 저장되지 않습니다.

<br />

## **4\. 기존 선 스타일을 복사하는 방법** [**시연 영상**](https://youtu.be/QDkYQaVkS5I)

1\) 좌측의 선 스타일 목록에서 복사하고자 하는 선 스타일 원본을 선택 후, 목록 아래 “복사하기” 버튼을 클릭합니다.

<br />

## **5\. CALS 선꼴 추가하기** [**시연 영상**](https://youtu.be/LxHtJXmzXh0)

1\) 좌측의 선스타일 하단, “건설 CALS 선꼴 확인” 을 클릭합니다.  
2\) 원하는 선꼴을 마우스 더블 클릭 시, 선 스타일에 해당 선 스타일이 추가됩니다.

<br />

## **6\. 외부 선꼴 추가하기** [**시연 영상**](https://youtu.be/0XvRl6ZKxWU)

1\) 좌측의 선스타일 하단, “건설 CALS 선꼴 확인”을 클릭합니다.  
2\) 상단에 카테고리 최우측에 ![image alt emoji][image7]버튼을 클릭합니다.  
3\) 선 스타일 파일을 로드 후, 해당 선 스타일을 더블 클릭 하면 선 스타일을 추가할 수 있습니다.

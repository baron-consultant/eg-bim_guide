- 도면에서 선택한 객체의 레이어를 다른 객체의 레이어와 동일하게 변경하는 기능입니다.

![][image1]

## **1\. 다른 객체와 동일하게 레이어 변경하기** [**시연 영상**](https://youtu.be/ePoPimAk9JY)

1\) 명령창에 “LAYMCH” 명령어를 입력합니다.  
2\) 레이어를 변경할 대상 객체를 선택한다.  
3\) 기준이 될 참조 객체를 선택한다.  
4\) 대상 객체의 레이어가 참조 객체의 레이어와 동일하게 변경된다.  
![][image2]

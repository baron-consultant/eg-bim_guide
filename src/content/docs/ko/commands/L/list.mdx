- 선택한 객체의 기본 속성 및 기하학적 정보(길이, 면적, 좌표 등)를 명령창에 상세하게 표시하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/jrckixd_Z2g)

1\) 명령창에 “LIST” 입력합니다.  
2\) 속성을 확인할 객체를 선택합니다.  
3\) ENTER 키를 누릅니다.  
4\) 명령창에 선택한 객체의 세부 정보가 출력됩니다.  
![][image2]

- 현재 도면에서 선택한 객체가 속한 레이어를 현재 레이어 로 변경하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/Px0IstllWCw)

1\) 명령창에 “LAYCUR” 명령어를 입력합니다.  
2\) 도면에서 원하는 객체를 선택한다.  
3\) 선택한 객체가 속한 레이어가 현재 레이어 로 설정된다.  
![][image2]

title: LAYFRZ

[image1]: ../../../../../assets/images/commands/layfrz/image1.png

- 선택한 객체가 속한 레이어를 동결 시켜 화면에서 보이지 않도록 하는 명령어입니다. 즉, 선택한 객체를 포함하는 레이어 전체가 잠시 “얼려진” 상태가 되어, 그 레이어의 모든 객체가 화면에서 사라지고 CAD의 성능 계산에서도 제외됩니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/RRkoYaJR2p0)

1\) 명령창에 “LAYFRZ” 명령어를 입력합니다.  
2\) 동결할 객체를 선택한다.  
3\) ENTER 키를 누른다.  
4\) 선택한 객체가 속한 레이어가 동결 되며, 해당 레이어의 모든 객체가 화면에서 사라집니다.

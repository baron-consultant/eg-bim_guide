[image1]: ../../../../../assets/images/commands/layuniso/image1.png

- LAYISO 명령으로 선택한 레이어만 표시된 상태에서, 숨겨지거나 잠긴 나머지 레이어들을 다시 원래 상태로 복원하는 기능입니다.

![][image1]

## **1\. 레이어 복원하기** [**시연 영상**](https://youtu.be/riwt-5s62cA)

1\) 명령창에 “LAYUNISO” 명령어를 입력합니다.  
2\) ENTER 키를 누릅니다.  
2\) LAYISO 실행 전 상태로 모든 레이어의 표시/잠금 상태가 복원된다. (숨겨졌던 레이어가 다시 표시되고, 잠겼던 레이어가 다시 해제됨)

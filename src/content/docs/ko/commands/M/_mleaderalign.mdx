* 여러 개의 멀티리더 객체를 수평 또는 수직으로 정렬하는 기능입니다.

![][image1]

## **1\. 지시선 정렬하기** [**시연 영상**]()

1\) 명령창에 “MLEADERALIGN” 를 입력합니다.  
2\) 정렬할 멀티리더(MLEADER) 객체들을 여러 개 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 명령창에 다음 옵션이 표시됩니다.

| 옵션 이름 | 기능 요약 | 설명 |
| :---: | :---: | :---: |
| Distribute | 균등 분배 | 선택한 리더 텍스트를 일정 간격으로 자동 정렬 |
| make leader segments Parallel | 리더선 평행 정렬 | 모든 리더선을 동일한 각도로 맞춤 |
| specify Spacing | 간격 수동 지정 | 리더 간 간격을 사용자가 직접 설정 |
| Use current spacing | 현재 간격 유지 | 기존 간격을 유지한 채로 기준선에 맞춰 정렬 |

5\) 원하는 정렬 방식을 선택합니다.  
6\) 마우스로 기준이 될 리더를 클릭하거나, 화면에서 정렬 방향을 지정합니다.  
7\) 선택한 모든 리더 텍스트가 지정한 기준에 따라 자동 정렬됩니다.

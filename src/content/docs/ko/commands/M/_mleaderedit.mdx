[image1]: ../../../../../assets/images/commands/mleaderedit/image1.png

* 기존 멀티리더 객체를 편집하는 기능으로 리더선의 추가, 삭제를 할 수 있습니다.

![][image1]

## **1\. 지시선 편집하기** [**시연 영상**]()

1\) 명령창에 “MLEADEREDIT” 를 입력합니다.  
2\) 수정할 멀티리더(MLEADER) 객체를 클릭합니다.  
3\) 다음과 같은 옵션들이 표시됩니다.

| 옵션 | 설명 |
| :---: | :---: |
| Add Leader (A) | 새로운 리더선을 추가합니다. |
| Remove Leader (R) | 기존의 리더선을 제거합니다. |

4\) 원하는 옵션을 입력하고 리더선을 클릭하면 추가, 삭제 됩니다.

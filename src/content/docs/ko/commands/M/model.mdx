- Layout(도면 공간) 에서 모델 공간(Model Space) 으로 전환하는 기능을 수행합니다.

![][image1]

## **1\. 모델 공간으로 전환하기** [**시연 영상**](https://youtu.be/sbT_PBSHJ4A)

1\) Layout 탭으로 이동합니다.  
2\) 명령창에 “MODEL” 명령어를 입력합니다.  
3\) ENTER 키를 누릅니다.  
4\) 도면 공간에서 모델 공간으로 전환됩니다.

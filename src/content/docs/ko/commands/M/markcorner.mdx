* 도면 내에서 도로의 갈매기차로를 자동으로 작성하는 명령어입니다.

![][image1]

## **1\. 갈매기차로 생성하기** [**시연 영상**](https://youtu.be/oO73ZkiawbE)

1\) 명령창에 “MARKCORNER” 를 입력합니다.  
2\) 첫 번째 차선을 선택합니다.  
3\) 두 번째 차선을 선택합니다.  
4\) 갈매기 차로의 기준 방법을 선택 합니다.

- P(두점) : 사용자가 직접 두 점을 지정하여 방향 결정  
- L(선객체) : 특정 선(Line)을 기준으로 방향 자동 인식

5\) 명령창의 안내에 따라 노면 표시 시작점과 끝점을 순서대로 지정합니다.

- 도면에서 선택한 객체를 새로운 레이어로 이동시키면서 동시에 해당 레이어를 생성하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/wgOiRBNpVQM)

1\) 명령창에 “MOVEENTNEWLAY” 명령어를 입력합니다.  
2\) 새 레이어로 이동할 객체를 선택한다.  
3\) 명령창에 새 레이어의 이름을 입력한다.

- 입력한 이름의 레이어가 없으면 자동으로 생성됨
- 이미 존재하는 레이어 이름을 입력하면 해당 레이어로 객체가 이동됨

4\) ENTER를 눌러 완료한다.  
![][image2]

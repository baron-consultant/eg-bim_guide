- 선택한 객체를 3차원에서 지정한 대칭 평면(Mirror plane) 을 기준으로 대칭 복사하는 기능입니다.

![][image1]

## **1\. 객체 2D 대칭 복사** [**시연 영상**](https://youtu.be/1KQuSU1tIhI)

1\) “MIRROR” 또는 “MI” 명령어를 실행합니다.  
2\) 대칭시킬 객체(선, 폴리선, 원, 사각형 등) 를 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 대칭 기준이 될 첫 번째 점을 지정합니다.  
5\) 대칭 기준이 될 두 번째 점을 지정하여 대칭축을 정의합니다.  
6\) 원본 객체를 유지할지 여부를 선택합니다.

- Yes → 원본 객체 삭제, 대칭된 객체만 남음
- No → 원본 객체 유지 \+ 대칭 복사본 생성

7\) ENTER 또는 SPACE BAR를 누릅니다.  
8\) 선택한 객체가 지정한 축을 기준으로 대칭 복사됩니다.

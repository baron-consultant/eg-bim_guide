![][image1]

## **1\. 객체 3D 대칭 복사** [**시연 영상**](https://youtu.be/KkdlAjMxHxs)

1\) “MIRROR3D” 또는 “3MI” 명령어를 실행합니다.  
2\) 대칭시킬 객체(3D 솔리드, 서피스, 3D 폴리선 등) 를 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 대칭 기준이 될 대칭 평면을 지정합니다.

- 세 점(3 points)으로 평면 지정
- 또는 X / Y / Z 평면 중 하나 선택 가능

5\) 대칭 후 원본 객체 유지 여부를 선택합니다.

- Yes → 원본 객체는 삭제되고 대칭된 객체만 남음
- No → 원본 객체 유지 \+ 대칭 복사본 생성

6\) ENTER 또는 SPACE BAR를 누릅니다.  
7\) 선택한 객체가 입력한 대칭 평면 기준으로 3차원 대칭됩니다.

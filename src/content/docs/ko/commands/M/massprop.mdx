* 3D 솔리드 또는 2D 영역의 체적을 계산하는 명령어입니다.

![][image1]

## **1\. 객체 체적 계산하기** [**시연 영상**](https://youtu.be/OmcC9-Wmhhs)

1\) 명령창에 “MASSPROP” 을 입력합니다.  
2\) 특성을 계산할 객체(3D 솔리드 또는 2D 영역) 을 선택합니다.  
3\) 엔터 키를 누릅니다.  
4\) 명령창에 객체 체적적 결과가 표시됩니다.

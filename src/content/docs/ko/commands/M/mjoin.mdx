* 떨어져 있는 선, 폴리선, 호 등을 클릭 지점을 기준으로 자동 연결선을 생성하여 하나의 연속된 폴리선으로 병합하는 기능입니다.

![][image1]

## **1\. 폴리선 병합하기** [**시연 영상**](https://youtu.be/wqUvrKj3fX0)

1\) 명령창에 “MJOIN” 를 입력합니다.  
2\) 병합할 객체(선, 호, 폴리선 등)를 선택합니다.  
3\) 두 선이 직접 연결되어 있지 않다면, 명령창 안내에 따라 간격이 있는 구간을 클릭합니다.  
4\) 클릭한 지점을 기준으로 자동 연결선이 생성됩니다.  
5\) 최종적으로 선택된 객체와 새 연결선이 하나의 폴리선으로 병합됩니다.

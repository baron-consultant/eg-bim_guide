title: MLEADERCOLLECT
---

[image1]: ../../../../../assets/images/commands/mleadercollect/image1.png

* 여러 개의 멀티리더 객체를 하나의 주석으로 자동 병합하는 기능입니다.

![][image1]

## **1\. 지시선 병합하기** [**시연 영상**]()

1\) 명령창에 “MLEADERCOLLECT” 를 입력합니다.  
2\) 같은 텍스트를 가진 멀티리더(MLEADER) 객체들을 모두 선택합니다.  
3\) 선택된 리더들이 하나의 주석 텍스트로 병합되고, 여러 개의 화살표가 해당 텍스트를 가리키는 형태로 자동 정리됩니다.  
    참고 : 다른 텍스트를 가진 리더는 병합되지 않습니다.

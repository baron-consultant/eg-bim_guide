* 객체 또는 두 점의 경사를 측정하고 작성합니다.

## **1\. 객체의 경사 표시** [**시연 영상**]()

1\) 명령창에 “MARKSLOPETEXT” 를 입력합니다.  
![][image1]  
2\) 팝업창에서, 적용방식, 경사 표시 형식, 경사도 문자 크기 등을 설정하고, ‘확인’ 을 클릭합니다.  
![][image2]  
3\) 도면에서 경사를 측정할 객체를 클릭하고, 엔터키를 누릅니다.  
![][image3]  
![][image4]  
![][image5]

## **2\. 두 점의 경사 표시** [**시연 영상**]()

![][image6]

![][image7]  
3\) 도면에서 경사를 측정할 두 점을 선택합니다.

- 선택한 객체를 사용자가 지정한 기준점에서 목표점으로 이동시키는 기능입니다.

![][image1]

## **1\. 객체 이동하기** [**시연 영상**](https://youtu.be/UR9erP-1wX4)

1\) “MOVE” 또는 “M” 명령어를 실행합니다.  
2\) 이동할 객체(선, 폴리선, 원, 블록, 치수선 등)를 선택합니다.  
3\) ENTER를 눌러 선택을 완료합니다.  
4\) 이동 기준이 될 첫 번째 점(기준점) 을 지정합니다.  
5\) 이동할 두 번째 점(목표점) 을 지정합니다.  
6\) ENTER 또는 SPACE BAR를 누릅니다.  
7\) 선택한 객체가 지정한 벡터만큼 이동됩니다.

* 외부 GIS(지리정보시스템) 데이터 파일을 도면으로 가져오는 명령어입니다.

![][image1]

## **1\. 외부 GIS 파일 가져오기** [**시연 영상**](https://youtu.be/4Bbm5poAEGg)

1\) 명령창에 “MAPIMPORT” 를 입력합니다.  
2\) 가져올 GIS 데이터 파일(SHP, SDF, SQLite 등)을 선택합니다.  
3\) 대화상자에서 필요한 파일을 선택합니다.  
![][image2]  
4\) “확인(OK)”을 눌러 도면에 데이터를 가져옵니다.

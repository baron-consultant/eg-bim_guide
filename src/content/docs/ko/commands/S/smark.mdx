- 단일 철근을 표기하는 기능입니다. 철근의 번호(RB_NO) 와 직경(RB_DIA) 값을 직접 입력하여, 도면 상에 자동으로 표기 텍스트를 생성합니다.

![][image1]

## **1\. 단일 철근 표기하기** [**시연 영상**](https://youtu.be/2QO4_hMcvzc)

1\) 명령창에 “SMARK” 를 입력합니다.  
2\) 표기를 삽입할 위치를 클릭합니다.  
3\) 명령창에서 표기 형식을 입력합니다.

- RB_NO 값 : 예) R1
- RB_DIA 값 : 예) D13

4\) 입력한 정보가 도면에 텍스트로 자동 삽입됩니다.  
![][image2]

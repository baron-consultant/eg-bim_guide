- 객체를 확대 / 축소하는 기능입니다.

![][image1]

## **1\. 객체 확대 / 축소** [**객체 축소 시연 영상**](https://youtu.be/bdctIM74BII) [**객체 확대 시연 영상**](https://youtu.be/fvnbObdD0jA)

1\) “SCALSE” 명령어를 실행합니다.  
2\) 크기를 조정할 객체(선, 폴리선, 원, 사각형 등) 을 선택합니다.  
3\) 기준점 을 지정합니다. (확대/축소 시 고정될 기준점)  
4\) 배율 을 입력합니다.

- \> 1 보다 큰 경우 확대 (예: 2 입력 → 2배 확대)
- \< 1 보다 작은 경우 축소 (예: 0.5 입력 → 절반 크기 축소)

5\) ENTER 또는 SPACE BAR를 누릅니다.  
6\) 선택한 객체가 입력한 배율에 맞게 확대/축소됩니다.

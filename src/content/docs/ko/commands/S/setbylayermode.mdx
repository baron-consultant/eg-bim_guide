- SETBYLAYER 명령어 실행 시 어떤 속성(Color, Linetype, Lineweight 등)을 ByLayer로 강제 변경할지 자동 적용 범위를 제어하는 시스템 변수입니다.

![][image1]

## **1\. 속성 설정하기**

1\) 명령창에 “SETBYLAYERMODE” 명령어를 입력합니다.  
2\) 원하는 값을 숫자로 입력합니다. (비트코드 방식)  
3\) 설정된 값에 따라 SETBYLAYER 명령어 실행 시 자동으로 적용되는 속성이 달라집니다.

## **2\. 값(비트 코드) 의미** [**시연 영상**](https://youtu.be/HaqCOT81Qog)

1\) 0 : 모든 옵션을 수동으로 지정 (기본값)  
2\) 1 : 색상(Color)을 ByLayer로 자동 설정  
3\) 2 : 선종류(Linetype)를 ByLayer로 자동 설정  
4\) 4 : 선가중치(Lineweight)를 ByLayer로 자동 설정  
5\) 8 : 투명도(Transparency)를 ByLayer로 자동 설정  
6\) 16 : 재질(Material)을 ByLayer로 자동 설정

여러 값을 동시에 적용하려면 합산해서 입력합니다.  
 예 : SETBYLAYERMODE \= 3 → 색상(1) \+ 선종류(2) 자동 적용

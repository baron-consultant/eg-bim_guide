title: SETBYLAYER
---

[image1]: ../../../../../assets/images/commands/setbylayer/image1.png

- 선택한 객체의 색상, 선종류, 선가중치 등 속성을 ByLayer 값으로 일괄 변경하는 기능입니다. 즉, 객체 개별 속성을 무시하고 해당 객체가 속한 레이어의 속성을 따르도록 강제 적용합니다.

![][image1]

## **1\. 객체 속성  ByLayer로 바꾸기** [**시연 영상**](https://youtu.be/sdP6eMB1jQc)

1\) 명령창에 “SETBYLAYER” 명령어를 입력합니다.  
2\) ByLayer 속성으로 변경할 객체를 선택합니다.  
3\) ENTER 키를 누릅니다.  
4\) 원하는 옵션(YES/NO)을 선택합니다.

- Color : 객체의 색상을 ByLayer로 변경
- Yes/No 옵션 : 블록 내부 객체도 변경할지 여부 선택

5\) ENTER를 누르면 선택한 옵션에 맞게 객체의 속성이 ByLayer로 적용됩니다.

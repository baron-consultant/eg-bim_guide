- 여러 개의 텍스트 객체를 연속으로 수정할 수 있는 기능입니다.

![][image1]

## **1\. 연속 텍스트 수정** [**시연 영상**](https://youtu.be/roEGatyxDP0)

1\) “SEQTEDIT” 명령어를 실행합니다.  
2\) SEQTEDIT 대화상자에서 필요한 옵션을 선택합니다.  
![][image2]  
3\) 수정할 여러 텍스트 객체를 선택합니다.  
4\) 선택한 텍스트가 하나씩 활성화되며, 사용자가 바로 내용을 수정할 수 있습니다.

- 각 텍스트의 편집을 마치고 ENTER 키를 누르면 다음 텍스트 편집으로 이동합니다.

5\) 모든 선택 텍스트 편집이 끝나면 ENTER 키로 종료합니다.  
![][image3]

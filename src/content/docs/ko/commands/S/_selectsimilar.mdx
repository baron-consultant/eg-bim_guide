- 도면에서 특정 조건(예: 레이어, 색상, 객체 종류 등)을 기준으로 객체를 빠르게 선택 하는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**]()

1\) 명령창에 “SELECTSIMILAR” 를 입력합니다.  
2\) SE를 입력하면 대화상자가 나타납니다.  
![][image2]  
3\) 대화상자에서 필요한 선택 조건을 설정합니다.  
4\) 도면에서 기준이 될 객체(하나)를 클릭합니다.  
5\) 동일한 속성을 가진 모든 객체가 자동으로 선택됩니다.  
 참고 : 선택된 객체들은 파란색 하이라이트로 표시됩니다.

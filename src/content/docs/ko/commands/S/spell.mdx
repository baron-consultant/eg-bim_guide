- 도면 내 텍스트 객체의 철자 검사를 수행하여 잘못된 철자를 확인하고 교정할 수 있는 기능입니다.

![][image1]

## **1\. 사용 방법** [**시연 영상**](https://youtu.be/cw6GpiWhPVs)

1\) 명령창에 “SPELL” 명령어를 실행합니다.  
2\) 철자 검사할 텍스트 객체를 선택 합니다.  
3\) ENTER 또는 SPACE BAR를 누르면 지정된 범위의 철자 검사가 완료됩니다.  
![][image2]

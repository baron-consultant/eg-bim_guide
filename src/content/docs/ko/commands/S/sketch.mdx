[image1]: ../../../../../assets/images/commands/sketch/image1.png

- 도면에 곡선을 자유롭게 그립니다.
- 곡선을 자유롭게 그리며 선 객체로 생성할 수 있습니다.

![][image1]

- SKETCH의 설정 옵션 키워드

* ENTER : 그려진 객체를 선으로 생성하고 종료합니다.
* Quit : 명령을 종료하여 스케치를 취소합니다.
* Pen down : 마우스로 클릭하여 그리기를 중단합니다.
* Pen up : 마우스로 클릭하여 그리기를 계속합니다.
* Erase : 선을 지웁니다.
* Write to drawing : 그려진 객체를 선으로 생성하고 이어서 그립니다.
* Connect : 그려진 객체를 연결합니다.
* Straight to cursor : 마우스 커서 위치로 직선을 그립니다.

## **1\. 도면에 자유롭게 그리고 선 객체 생성하기 \- ENTER** [**시연 영상**](https://youtu.be/AiE43CLWZ9U)

1\) “SKETCH” 명령어를 실행합니다.  
2\) Length of segments 값을 설정합니다.(정밀한 스케치가 필요하면 작은 값 / 대략적인 스케치만 필요하면 큰값)  
3\) 마우스로 클릭하여 자유롭게 그립니다.  
4\) ENTER워드를 입력해서 그려진 상태의 선 객체를 생성하고 명령어를 종료합니다.

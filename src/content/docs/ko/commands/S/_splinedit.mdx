title: SPLINEDIT
---

[image1]: ../../../../../assets/images/commands/splinedit/image1.png

- 스플라인 객체를 편집하는 명령어입니다.
- 이 명령어를 통해 이미 생성된 스플라인의 다양한 속성을 수정할 수 있습니다.

![][image1]

- 옵션 키워드

* Fit data : 스플라인의 맞춤점을 조정합니다.  
  (Add / Close / Delete / Move / Purge / Tangents / toLerance)
* Close/Open : 스플라인을 닫거나 엽니다.
* Move vertex : 스플라인의 제어점을 이동시킵니다.  
  (Next / Previous / Select Point)
* Refine : 스플라인의 세부적인 속성을 조정합니다.  
  (Add control point / Elevate order / Weight)
* rEverse : 방향을 반대로 변경합니다.
* convert to Polyline : 스플라인을 폴리라인으로 변환합니다.
* Undo : 마지막 편집 작업을 취소합니다.

##

## **1\. 스플라인의 새로운 점 추가하기 \- Fit Data_Add**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Add(A) 옵션 키워드를 입력합니다.  
4\) 스플라인의 시작점 or 끝점을 마우스로 클릭합니다.  
5\) 새로운 점을 클릭하거나 After / Before 옵션 키워드를 입력합니다.  
 5-a) 새로운 점을 원하는 지점에 마우스로 클릭 하다가 eXit 키워드로 명령어를 종료합니다.  
 5-b) After 옵션 : 선택한 점 이후에 새로운 점을 추가합니다. (선택한 점과 그 다음 점 사이에  
 새 점이 삽입)  
 5-c) Before 옵션 : 선택한 점 이전에 새로운 점을 추가합니다. (선택한 점과 그 이전 점 사이에  
 새 점이 삽입)  
6\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **2\. 스플라인 객체 닫기 열기- Fit Data_Close Open**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Close(C) 옵션 키워드를 입력합니다.  
 3-a) 객체가 닫혀있다면 Open(O) 키워드가 활성화됩니다.  
 3-b) 객체가 열려있다면 Close(C) 키워드가 활성화됩니다.  
4\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **3\. 스플라인 맞춤점 제거하기 \- Fit Data_Delete**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Delete(D) 옵션 키워드를 입력합니다.  
4\) 제거할 맞춤점을 클릭합니다.  
5\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **4\. 스플라인 맞춤점 이동하기 \- Fit Data_Move**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Move(M) 옵션 키워드를 입력합니다.  
4\) 이동시키고자 하는 맞춤점을 선택 후, 이동시킬 위치에 마우스로 클릭합니다.  
 4-a) Next 옵션 : 다음 맞춤점을 이동시킬때 사용  
 4-b) Previous 옵션 : 이전 맞춤점을 이동시킬때 사용  
 4-c) Select Point 옵션 : 직접 맞춤점을 선택할때 사용  
5\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **5\. 맞춤점 데이터를 제어점으로 대체하기 \- Fit Data_Purge**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Purge(P) 옵션 키워드를 입력합니다.  
4\) SPACE BAR 또는 ENTER를 입력하여 명령어를 종료합니다.

## **6\. 스플라인의 시작점과 끝점의 접선방향 변경하기** **\- Fit Data_Tangents**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 Tangents(T) 옵션 키워드를 입력합니다.  
4\) 시작점을 마우스로 클릭합니다.  
5\) 끝점을 마우스로 클릭합니다.  
6\) SPACE BAR 또는 ENTER를 입력하여 명령어를 종료합니다.

## **7\. 스플라인의 공차 설정하기 \- Fit Data_toLerance**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Fit data(F) 키워드를 입력하여 toLerance(L) 옵션 키워드를 입력합니다.  
4\) 공차값을 입력합니다.  
5\) SPACE BAR 또는 ENTER를 입력하여 명령어를 종료합니다.

## **8\. 스플라인 닫기 \- Close**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Close(C) 옵션 키워드를 입력합니다.  
 3-a) 닫혀있는 스플라인의 경우 Open(O) 옵션 키워드를 입력합니다.  
5\) SPACE BAR 또는 ENTER를 입력하여 명령어를 종료합니다.

## **9\. 스플라인의 제어점 이동하기 \- Move vertex**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) 이동시키고자 하는 제어점을 선택 후, 이동시킬 위치에 마우스로 클릭합니다.  
 3-a) Next 옵션 : 다음 제어점을 이동시킬때 사용  
 3-b) Previous 옵션 : 이전 제어점을 이동시킬때 사용  
 3-c) Select Point 옵션 : 직접 제어점을 선택할때 사용  
4\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **10\. 스플라인을 정밀하게 조정하기 \- Refine**

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.  
3\) Refine(R) 옵션 키워드를 입력합니다.  
4\) Add control point / Elevate order / Weight 옵션 키워드를 입력합니다.  
 4-a) Add control point 옵션 : 스플라인에 마우스 클릭으로 새로운 제어점을 추가합니다.  
 4-b) Elevate order 옵션 : 차수를 입력하여 더 부드럽고 유연한 곡선을 만들 수 있습니다.  
 4-c) Weight 옵션 : 제어점의 가중치를 입력합니다.(N / P / S 옵션으로 변경할 제어점 선택)  
5\) eXit(X) 키워드 또는 SPACE BAR를 입력하여 명령어를 종료합니다.

## **11\. 스플라인의 방향 반전시키기 \- rEverse**

1\) 명령어 실행 전에 속성창에서 스플라인 객체의 넘버를 확인합니다.  
2\) “SPLINEDIT” 명령어를 실행합니다.  
3\) 스플라인 객체를 선택합니다.  
4\) rEverse(E) 옵션 키워드를 입력합니다.  
5\) SPACE BAR 또는 ENTER를 입력하여 명령어를 종료합니다.  
6\) 속성창에서 스플라인 객체가 반전 되었는지 확인합니다.

## **12\. 스플라인을 폴리선으로 변환하기 \- convert to Polyline** [**시연 영상**]()

1\) “SPLINEDIT” 명령어를 실행합니다.  
2\) 스플라인 객체를 선택합니다.

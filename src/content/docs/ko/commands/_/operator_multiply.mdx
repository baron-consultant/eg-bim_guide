* 도면 내에 존재하는 숫자가 포함된 텍스트 객체들의 값을 곱하기 계산하는 기능입니다.

![][image1]

## **1\. 사용 방법**

1\) 명령창에 “\*” 를 입력합니다.  
2\) 명령창 안내에 따라 숫자가 포함된 텍스트 객체들을 순서대로 선택합니다.  
3\) 선택이 끝나면 엔터 키를 누릅니다.  
4\) 삽입할 지점을 클릭하면 곱하기 결과가 도면에 텍스트로 추가됩니다.

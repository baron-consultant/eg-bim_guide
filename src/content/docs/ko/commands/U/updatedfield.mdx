- 도면에 삽입된 필드의 값을 자동으로 업데이트할 타이밍을 설정하는 기능입니다.

![][image1]

## **1\. 필드 갱신** [**시연 영상**](https://youtu.be/2yJSPuNdKXI)

1\) “UPDATEFIELD” 명령어를 실행합니다.  
2\) 갱신할 대상을 지정합니다.  
3\) 선택된 필드가 최신 값으로 업데이트됩니다.

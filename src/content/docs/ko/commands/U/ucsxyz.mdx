- 현재 사용자 좌표계의 X, Y, Z 축을 순차적으로 90도씩 회전시키는 기능입니다.

![][image1]

## **1\. 축 기준 UCS회전** [**시연 영상**](https://youtu.be/NLhgCJrP51E)

1\) 명령창에 “UCSXYZ” 를 입력한다.  
2\) 현재 UCS가 X축을 기준으로 90도 회전한다.  
3\) 명령어를 한 번 더 입력하면 Y축 기준으로 90도 회전한다.  
4\) 세 번째로 입력하면 Z축 기준으로 90도 회전한다.  
5\) 이후 다시 입력 시 X → Y → Z 순환 반복으로 회전한다.

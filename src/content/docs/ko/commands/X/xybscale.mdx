- 선택한 객체를 X축, Y축을 기준으로 각각 다른 비율로 확대 / 축소하는 기능입니다.

![][image1]

## **1\. 객체 확대 / 축소 (X, Y축 개별 배율)** [**객체 확대 시연 영상**](https://youtu.be/qqP28tgr3sI) [**객체 축소 시연 영상**](https://youtu.be/Wy6Y-Bcpelw)

1\) “XYB” 또는 “XYBSCALE” 명령어를 실행합니다.  
2\) X, Y축 배율을 입력 합니다

- \> 1 보다 큰 경우 확대 (예: 2 입력 → 2배 확대)
- \< 1 보다 작은 경우 축소 (예: 0.5 입력 → 절반 크기 축소)

3\) 크기를 조정할 객체(선, 폴리선, 원, 사각형 등)을 선택합니다  
4\) 기준점 을 지정합니다. (확대/축소 시 고정될 기준점)  
5\) 선택한 객체가 입력한 배율에 맞게 확대/축소됩니다.

* 외부참조(Xref) 객체를 도면 내에서 직접 편집할 수 있는지 여부를 설정하는 시스템 변수 입니다.

![][image1]

## **1\. 외부참조 편집 기능 설정하기** [**시연 영상**]()

1\) 명령창에 “XEDIT” 을 입력합니다.  
2\) 아래 중 하나의 값을 입력합니다.

| 입력값 | 설명 |
| :---: | ----- |
| OFF | 외부참조(Xref)를 도면 내에서 편집할 수 없습니다. 즉, REFEDIT 명령 또는 더블 클릭 편집이 불가능 |
| ON | 외부참조(Xref)를 도면 내에서 직접 편집할 수 있습니다. 즉, REFEDIT 명령 사용 가능 (기본값). |

3\) 엔터 키를 눌러 적용합니다.

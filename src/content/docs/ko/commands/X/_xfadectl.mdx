title: XFADECTL

[image1]: ../../../../../assets/images/commands/xfadectl/image1.png

* 편집 중인 참조 화면에서 외부참조(Xref) 또는 블록(Block) 의 배경 흐림(페이드, Fade) 정도를 제어하는 시스템 변수입니다.

![][image1]

## **1\. 편집 중 배경 흐림(페이드) 설정하기** [**시연 영상**]()

1\) 명령창에 “XDWGFADECTL” 을 입력합니다.  
2\) 표시 강도를 0 \~ 90 사이의 값으로 입력합니다.  
3\) 입력한 값이 클수록 외부참조의 표시가 더 흐릿해집니다.

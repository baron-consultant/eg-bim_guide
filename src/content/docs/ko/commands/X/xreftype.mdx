- 외부 참조(Xref)의 삽입 방식을 지정하는 기능입니다. Xref를 도면에 첨부할 때 첨부 또는 중첩 방식 중 하나를 선택할 수 있습니다.

![][image1]

## **1\. 외부 파일 삽입 방식 설정하기** [**시연 영상**](https://youtu.be/vs68kTlWJ38)

1\) 명령창에 “XREFTYPE” 명령어를 실행합니다.  
2\) 원하는 옵션을 입력합니다.

- OFF : 외부 참조를 일반 첨부로 삽입(Attach 방식)  
  다른 도면에서 이 도면을 다시 Xref로 불러올 경우, 함께 따라옴.
- ON : 외부 참조를 중첩으로 삽입(Overlay 방식)  
  다른 도면에서 이 도면을 다시 Xref로 불러와도, 중첩된 Xref는 따라오지 않음.

3\) ENTER 또는 SPACE BAR를 누르면 설정이 적용됩니다.  
4\) 이후 삽입되는 Xref는 지정한 기본 방식(Attach 또는 Overlay)으로 적용됩니다.

* 도면에 삽입된 외부참조(Xref) 도면의 표시 투명도를 제어하는 시스템 변수입니다.

![][image1]

## **1\. 외부참조 희미함(페이드) 설정하기** [**시연 영상**]()

1\) 명령창에 “XDWGFADECTL” 을 입력합니다.  
2\) 표시 강도를 0 \~ 90 사이의 값으로 입력합니다.  
3\) 입력한 값이 클수록 외부참조의 표시가 더 흐릿해집니다.  
    참고 :  값 변경 후 화면에 즉시 반영되지 않을 경우, "REGEN" 명령으로 화면을 새로 고침합니다.
